Problem G — Maximum-Likelihood Decoding

The weight of a binary word is the number of ones in it. The distance between two binary words, d(a,b), is the number of positions in which they differ. The minimum distance of a set of binary words is the smallest distance between any two distinct words in that set.

  1. Verify that each codeword in the following set, C1, satisfies the parity equations a4=a1+a3 and a5=a1+a2+a3.

    Codeword a1+a3 a1+a2+a3
    00000 0 0
    00111 1 1
    01001 0 1
    01110 1 0
    10011 1 1
    10100 0 0
    11010 1 0
    11101 0 1

  2. Define a code, C2 in B6, with the first three bits being information, and the remainder satisying: a4=a2, a5=a1+a2, and a6=a1+a2+a3.

    1. List the codewords in C2:

      000000, 001001, 010111, 011110, 100011, 101010, 110100, 111101

    2. Find the minimum distance between codewords in C2.

      The distance between 000000 and 001001 (and some other pairs) is two. Since the information part of every codeword differs from that of every other codeword, and the parity part of each codeword also differs from that of every other codeword, no smaller difference is possible.

    3. How many errors in any word of C2 are guaranteed to to be detected?

      This code is only guaranteed to detect single errors, since there are pairs of codewords that only differ in two positions.

  3. Design a code in B4, with two positions for information and two for parity.

    Let a3=a1 and a4=a2. The codewords are then 0000, 0101, 1010, and 1111. The minimum distance is 2.

  4. Find the words in C1 closest to each of the following:

    1111111101
    0010100111
    1100011010
    1001111101
    1000110011
    1011100111, 10011

  5. For a code in Bn with a minimum distance of m, prove that it is always possible to detect up to m-1 transmission errors in a single word.

    Since any two words, a and b, will always differ in m bits, an error involving no more than m-1 bits will change the transmitted word into something that differs from any word by at least one bit.

  6. Define the sphere of radius k about the word a to be the set of all words in Bn such that their distance from a is no greater than k. S(a;k) = {x∈Bn : d(a,x)≤k}. If m is the minimum distance for some set of binary words, and t=(m-1)/2, show that any two spheres of radius t have an empty intersection.

    First, I will show that d, as previously defined, is a metric function.

    • d(a,a) = 0 — This is true, since a binary word doesn't differ in any positions from itself.
    • d(a,b) > 0 when a≠b — Since a and b must differ in at least one position to be different words, this is also true.
    • d(a,b) = d(b,a) — The number of bits that differ between a and b is the same no matter which we look at first.
    • d(a,b)+d(b,c) ≥ d(a,c) — If none of the positions in which a and b differ are the same as any of the positions in which c and b differ, then d(a,b)+d(b,c) = d(a,c). Otherwise, we have the case where a and c both differ from b in the same position at least once. In any of these positions, a and c then have the same bits, so their distance is reduced by the number of positions of overlap.

    Therefore, d is a metric function.

    Let a and c be words in Bn, with a≠c. Assume that the two spheres, S(a;t) and S(c;t), have an non-empty intersection. Let b ∈ S(a;t)∩S(c;t). Then, since b∈S(a;t), d(a,b)≤t. Similarly, d(b,c)≤t. By the triangle inequality, d(a,c)≤t+t. t+t = 2t = 2(m-1)/2 = m-1. Since the minimum distance between two distinct words is m, the assumption that S(a;t) and S(c;t), have an non-empty intersection is false.

    Another way to show this is to notice that, since (m-1)/2 is less than m, any sphere, S(x;t), contains only one element, x.

  7. Use the fact just shown to prove that in any transmission of a codeword that has no more than t transmission errors, the codeword can be correctly decoded.

    This follows immediately from the fact that any sphere, S(x;t), contains only one element, x.

  8. Show that any transmission of a codeword in C2 with one error can be corrected and any transmissions of two errors can be detected.

    Since the minimum distance of C2 is 2, t = (m-1)/2 = 1/2. Therefore, the claim is false. A received code word of 000001 can be detected as incorrect, but it is not possible to tell whether it was supposed to be 000000 or 001001, either of which differs from it in only one position.
