Problem D — Elementary Properties of Cyclic Subgroups of Groups

Let G be a group and let a,b ∈ G. Prove the following:

  1. If a is a power of b, say a = bk, <a> ⊆ <b>.

    <a> is the set of all powers of a, each of which is also a power of b. Since <b> is the set of all powers of b, <a> ⊆ <b>.

  2. Suppose a is a power of b, say a = bk, <a>. Then b is a power of a iff <a> = <b>.

    If b ∈ <b>. If <a> = <b>, then b ∈ <a>. Since <a> is the set of powers of a, this means that b is a power of a.

    Only If If b is a power of a, then every power of b is a power of a. This means that <b> ⊆ <a>. As shown in Exercise 1, <a> ⊆ <b>. Therefore <a> = <b>.

  3. Suppose a ∈ <b>. Then <a> = <b> iff ord(a) = ord(b).

    If we allow infinite groups, it is true that 2 ∈ <1> (with addition as the group operation), and ord(2) = ord(1) but <2> ≠ <1>.

    If By Exercise 1, a ∈ <b> implies that <a> ⊆ <b>. The order of a group generated by an element is the same as the order of the element. If ord(a) = ord(b), then |<a>| = |<b>|. If we make the assumption that <b> is a finite group (something that Pinter does not state), the facts that <a> ⊆ <b> and |<a>| = |<b>| combine to give <a> = <b>.

    Only If So, if ord(a) ≠ ord(b), then |<a>| ≠ |<b>|. Since <a> and <b> are sets, this means that <a> ≠ <b>.

  4. Let ord(a) = n, and let b = ak. Then <a> = <b> iff n and k are relatively prime.

    If By Exercise C1, if n and k are relatively prime, ak generates <a>. Since b = ak, this means that b generates <a>, or <a> = <b>

    Only If If n and k are not relatively prime, then lcm(n,k)<nk. Then, ord(b)<n and |<b>| < |<a>|, which means that <a> ≠ <b>.

  5. Let ord(a) = n, and suppose a has a kth root, a = bk. Then <a> = <b> iff n and k are relatively prime.

    The same as the previous exercise.

  6. Any cyclic group of order mn has a unique subgroup of order n.

    Let a be the generator of a group of order mn. Then { akm : k∈N } is such a subgroup. This is obvious from the fact that ord(am) = n.
