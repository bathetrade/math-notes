Problem E — Isomrphism of Infinite Groups

  1. Let E designate the group of all even integers, with respect to addition. Prove that Z ≘ E.

    Let f:Z → E be f(p) = 2p.

    This function is injective. Assume that f(a) = f(b) for some a,b ∈ Z. Then 2a = 2b, and a = b. This function is surjective. Let t ∈ E. Since t is even, t/2 ∈ Z, and f(t/2) = 2t/2 = t. Since this function is injective and surjective, it is a bijection.

    Let x,y ∈ Z. Then f(x)f(y) = 2x + 2y = 2(x+y) = f(x+y). This function preserves "products".

    Since this is a bijection that preserves products, it is an isomorphism between Z and E. Therefore Z and E are isomorphic.

  2. Let G be the group {10n : n ∈ Z} with respect to multiplication. Prove that G ≘ Z.

    I don't have to prove this, do I?

    f:Z → G such that f(v) = 10v is obviously an injection from Z to G.

    Let p,q ∈ Z. Then f(p)f(q) = 10p10q = 10p+q = f(p+q). Therefore, f preserves products.

    Since there is a bijection from Z to G that preserves products, Z and G are isomorphic.

  3. Prove that C ≘ R×R.

    I'm going to assume that he's using "C" to refer to the group of complex numbers with respect to addition.

    f:R×R → C such that f(a,b) = a + ib is a bijection.

    Let (p,q) and (r,s) ∈ R×R. Then f(p,q)f(r,s) = (p+iq) + (r+is) = (p+r) + i(q+s) = f((p+r),(q+s)), and f preserves products.

    R×R is therefore isomorphic with C.

  4. Prove that R is not isomorphic to R* (the multiplicative group of non-zero reals).

    In R*, -1×-1 = 1. This is an element other than the identity that is its own inverse.

    In R, an element that is not the identity is either positive or negative. Adding a positive number to itself gives another positive number. Adding a negative number to itself gives another negative number. Therefore, there cannot be any element of R (other than the identity) that is its own inverse.

    Therefore, R and R* are not isomorphic.

  5. Prove that Z is not isomorphic to Q.

    Let f:Z → Q such that f(x) + f(y) = f(x+y). Since 1 ∈ Z, f(1) ∈ Q.

    Let f(1) = b. Since b ∈ Q, b/2 ∈ Q. f(1) = b = b/2 + b/2. Assume that there is a ∈ Z such that f(a) = b/2. This gives us f(a) + f(a) = b/2 + b/2 = b = f(1).

    Since f was specified to preserve products, this means that f(a+a) = f(a) + f(a) =f(1), which means that a+a = 1. Since there is no element of Z that gives 1 when added to itself, a ∉ Z, which means that f is not surjective, not bijective, not an isomorphism.

  6. Prove that Q is not isomorphic to Qpos (the multiplicative group of positive rationals).

    Let f:Q → Qpos preserve products. That is, if p,q ∈ Q, f(p+q) = f(p)f(q).

    Let x ∈ Q such that f(x) = 2. Since x ∈ Q, x/2 ∈ Q. f(x/2)f(x/2) = f(x/2 + x/2) = f(x) = 2. If f(x/2)f(x/2) = 2, then f(x/2) = √2.

    Since Qpos ⊂ Q, and √2 ∉ Q, √2 ∉ Qpos. Therefore, there is no x ∈ Q such that f(x) = 2, and f is not surjective, not bijective, not an isomorphism.
