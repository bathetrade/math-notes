Problem B — Functions on R and Z

For each of the functions in exercises 1-4, determine whether it is injective or surjective.

  1. f:R→(0,∞), defined by f(x) = ex.

    This function is injective. Let f(a)=f(b) for some a, b. Then:

    f(a) f(b) given
    ea eb definition of f
    log(ea) log(eb) take log of both sides
    a b log is inverse of exponential

    This function is surjective. Let y∈(0,∞). Then, log(y)∈R and elog(y)=y.

  2. f:(0,1)→R, defined by f(x) = tan(x).

    I find my answers to the first two exercises unsatisfying, since they rely on claiming that these functions have inverses. This is only possible when they're injective.

    This function is injective. Let f(a)=f(b) for some a,b. Then:

    f(a) f(b) given
    tan(a) tan(b) definition of f
    arctan(tan(a)) arctan(tan(b)) take arctan of both sides
    a b arctan is inverse of tangent

    This function is not surjective. Its domain is roughly (0,1.5574...). It does not map to any number outside of this range.

  3. f:R→Z, defined as the least integer greater than or equal to x.

    This function is not injective, since any element of [1,2) maps to 1.

    This function is surjective. Let y∈Z. Then, f maps any element of [y,y+1) to y.

  4. f:R→Z, defined by

    f(n) n+1 n is even
    n-1 n is odd

    First, we note that f maps even integers to odd integers and it maps odd integers to even ones.

    This function is injective. If f(p)=f(q) is odd, p=f(p)-1=f(q)-1=q. If f(p)=f(q) is even, p=f(p)-1=f(q)-1=q.

    This function is surjective. If k is odd, f(k-1)=k. If k is even, f(k+1)=k.

  5. Find a bijective function f:Z→E.

    f(n) = 2n will do nicely.

    If f(p)=f(q), then p=f(p)/2=f(q)/2=q, so f is injective.

    If q∈E, then (q/2)∈Z and f(q/2)=q, so f is surjective.

    Since f is both injective and surjective, it is bijective.
