Problem C — Equivalence Relations and Partitions of R × R

Prove that each of the following is a partition or equivalence relation, describe it geometrically and give the corresponding equivalence relation or partition.

  1. For each r ∈ R, Ar = {(x,y) : y = 2x + r}.

    Let (a,b) ∈ R × R. Let c = b - 2a. Then, b = 2a + c, and (a,b) ∈ Ac. Therefore, any element of R × R is in Ar for some r in R.

    Assume that (a,b) ∈ Ad. Then, b = 2a + d, or d = b - 2a = c. Thus, every element of R × R is in only one Ar.

    Therefore, {Ar : r ∈ R} is a partition of R × R.

    The partition is the set of all lines in the Cartesian plane that have a slope of 2. They are distinguished by their y-intercepts.

    The equivalence relation is (x1,y1)~(x2,y2) iff (y1-y2) = 2(x1-x2).

  2. For each r ∈ R, Ar = {(x,y) : x2 + y2 = r2}.

    Let (a,b) ∈ R × R. Let c = sqrt(a2 + b2). Then, a2 + b2 = c2, and (a,b) ∈ Ac. Therefore, any element of R × R is in Ar for some r in R+ ∪ {0}.

    Since (-r)2 = r2, A-r = Ar.

    Assume that (a,b) ∈ Ad. Then d2 = a2 + b2 = c2, or d2 = c2. In R, this is only true when d = ±c. Thus, every element of R × R is in only one Ar.

    Therefore, {Ar : r ∈ R} is a partition of R × R.

    The partition is the set of all circles in the Cartesian plane centered on the origin. They are distinguished by their radii.

    The equivalence relation is (x1,y1)~(x2,y2) iff x12 + y12 = x22 + y22.

  3. For each r ∈ R, Ar = {(x,y) : y = |x| + r}.

    Let (a,b) ∈ R × R. Let c = b - |a|. Then, b = |a| + c, and (a,b) ∈ Ac. Therefore, any element of R × R is in Ar for some r in R.

    Assume that (a,b) ∈ Ad. Then, b = |a| + d, or d = b - |a| = c. Thus, every element of R × R is in only one Ar.

    Therefore, {Ar : r ∈ R} is a partition of R × R.

    The partition is the set of all vee-shapes in the Cartesian plane that have their minima at the y-axis and slopes of ∓1. They are distinguished by their minima (which are also their y-intercepts).

    The equivalence relation is (x1,y1)~(x2,y2) iff (y1-y2) = (|x1|-|x2|).

  4. (x,y)~(u,v) iff ax2 + by2 = au2 + bv2. (a,b>0)

    Reflexive Since ax2 + by2 = ax2 + by2, this is reflexive.

    Symmetric If ax2 + by2 = au2 + bv2, then au2 + bv2 = ax2 + by2, so this is symmetric.

    Transitive If ax2 + by2 = au2 + bv2 and au2 + bv2 = ap2 + bq2, then ax2 + by2 = ap2 + bq2, so this is transitive.

    Since it is reflexive, symmetric, and transitive, it is an equivalance relation.

    The partition is the set of all ellipses in the Cartesian plane with the same ratio of major to minor axis, centered on the origin, with the major axis coincident with one of the coordinate axes (the same axis for all of the ellipses). They are distinguished by the length of their major axis (or their minor axis).

    For each r ∈ R+ ∪ {0}, Ar = {(x,y) : ax2 + by2 = r2}.

  5. (x,y)~(u,v) iff x+y = u+v.

    Reflexive Since x+y = x+y, this is reflexive.

    Symmetric If x+y = u+v then u+v = x+y, so this is symmetric.

    Transitive If x+y = u+v and u+v = p+q then x+y = p+q, so this is transitive.

    Since it is reflexive, symmetric, and transitive, it is an equivalance relation.

    The partition is the set of all lines in the Cartesian plane with a slope of -1. They are distinguished by their y-intercepts.

    For each r ∈ R, Ar = {(x,y) : x+y = r}.

  6. (x,y)~(u,v) iff x2 - y = u2 - v.

    Reflexive Since x2 - y = x2 - y, this is reflexive.

    Symmetric If x2 - y = u2 - v, then u2 - v = x2 - y, so this is symmetric.

    Transitive If x2 - y = u2 - v and u2 - v = p2 - q, then x2 - y = p2 - q, so this is transitive.

    Since it is reflexive, symmetric, and transitive, it is an equivalance relation.

    The partition is the set of all parabolas similar to y = x2, centered on the y-axis. They are distinguished by their y-intercepts.

    For each r ∈ R, Ar = {(x,y) : y = x2 - r}.
