Problem H — Some General Properties of Isomorphism

  1. Let G and H be groups. Prove that G × H ≘ H × G.

    Let IGH:G × H → H × G be defined as IGH(g,h) = (h,g). If a,c∈G and b,d∈H, then IGH(a,b) = (b,a) = (d,c) = IGH(c,d) if and only if a=c and b=d. Therefore, IGH is a bijection.

    IGH(a,b)*IGH(c,d) = (b,a)*(d,c) = (bd,ac) = IGH(ac,bd) = IGH((a,b)(c,d)). Therefore, IGH preserves products.

    Since IGH is a bijection that preserves products, it is an isomporphism, and its domain and co-domain are isomorphic.

  2. If G1 ≘ G2 and H1 ≘ H2, prove that G1 × H1 ≘ G2 × H2.

    Let g:G1 → G2 and h:H1 → H2 be isomorphisms. Define IGH:G1 × H1 → G2 × H2 as IGH(p,q) = (g(p),h(q)).

    Since g and h are bijections, IGH(a,b) = (g(a),h(b)) = (g(c),h(d)) = IGH(c,d) if and only if a=c and b=d. Therefore, IGH is a bijection.

    Since g and h are isomorphisms, IGH(a,b)IGH(c,d) = (g(a),h(b))(g(c),h(d)) = (g(a)g(c),h(b)h(d)) = (g(ac),h(bd)) = IGH((a,b)(c,d)).

    Since IGH is a bijection that preserves products, it is an isomorphism. Therefore, its domain and codomain are isomorphic.

  3. Let G be any group. Prove that G is Abelian if and only if f(x) = x-1 is an isomorphism from G to G.

    Since every element in a group has a unique inverse, f is a bijection.

    To show that f preserves products, we will make use of the fact that (ab)-1 = b-1a-1.

    If G is Abelian, then rs = sr for all r,s∈G. f(rs) = (rs)-1 = (sr)-1 = r-1s-1 = f(r)f(s). Therefore, if G is Abelian, f preserves products and is an isomorphism.

    f(pq) = (pq)-1 = q-1p-1 = f(q)f(p). If f is an isomorphism, then it preserves products, so it must be the case that f(pq) = f(p)f(q). Therefore, f(p)f(q) = f(q)f(p), and G is shown to be Abelian.

  4. Let G be any group, with its operation expressed multiplicatively. Let H be a group with the same set as G, and let its operation be x*y = y·x, where "·" is the operation of G. Prove that G ≘ H.

    I will show that f:G → H, defined as f(x) = x-1, is an isomorphism.

    As shown in Exercise 3, f is a bijection.

    f(p·q) = (p·q)-1 = q-1·p-1 = p-1*q-1 = f(p)*f(q)

    Therefore, f is preserves products and is an isomorphism. This provess that G ≘ H.

  5. Let c be a fixed element of G. Let H be a group with the same set as G and the operation x*y = xcy. Prove that the function f(x) = c-1x is an isomorphism from G to H.

    This was probably proved earlier, but I'm too lazy to find the proof.

    For any element p∈H, f(cp) = p, so f is onto H. For any elements a,b∈G, if f(a)=f(b), then c-1a=c-1b, cc-1a=cc-1b, and a=b. Therefore, f is one-to-one and a bijection.

    f(rs) = c-1rs = c-1res = c-1rcc-1s = f(r)cf(s) = f(r)*f(s). Therefore f preserves products.

    Since f is a bijection that preserves products, it is an isomorphism.
