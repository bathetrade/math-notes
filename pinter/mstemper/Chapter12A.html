Problem A — Examples of Partitions

Prove that each of the following is a partition of the indicated set. Then describe the equivalence relation associated with that partition.

  1. For each integer r ∈ {0, 1, 2, 3, 4}, let Ar be the set of all integers that leave a remainder of r when divided by 5. Prove that {A0, A1, A2, A3, A4} is a partition of Z.

    That every integer leaves a remainder in {0, 1, 2, 3, 4} when divided by 5 follows from the Division Algorithm, as does the fact that the remainder for any integer is unique. Therefore, every integer is in exactly one of these sets.

    Since {0, 1, 2, 3, 4} each is its own remainder when divided by 5, every element of {Ar : r ∈ {0, 1, 2, 3, 4}} has at least one element.

    Since every one of these sets has at least one element and every integer is in exactly one of these sets, it is a partition.

    The equivalence relation is p~q iff p ≅ q (mod 5).

  2. For each integer n, let An = {x∈Q : n≤x<n+1}. Prove {An : n ∈ Z} is a partition of Q.

    Let q ∈ Q. We have two possibilities: q ∈ Z or q ∉ Z.

    If q ∈ Z, then there is exactly one n such that n≤q<n+1, n=q. We know this because the integers are distinct and n<n+1.

    Furthermore, for every integer j, Aj contains j. Therefore, every member of {An : n ∈ Z} has at least one element.

    If q ∉ Z, then for each integer i, it is either true that i<q or q<i, by trichotomy. Let p be the least integer greater than q. This must exist, and be unique, by the Well-Ordering Principle. Let n+1 = p, which gives us a unique n such that n≤q<n+1.

    So for every rational x, there is a unique n such that x ∈ An, and {An : n ∈ Z} is a partition.

    The equivalence relation is defined by the floor function: p~q iff floor(p)=floor(q).

  3. For each rational number r, let Ar = {(m,n) ∈ Z×Z* : m/n=r}. Prove that {Ar : r∈Q} is a partition of Z×Z*.

    Every rational number may be represented as an element of {(m,n) ∈ Z×Z* : m/n=r}, by the definition of "rational number". Therefore, every member of {Ar : r∈Q} has at least one element.

    Any ordered pair of integers (the second one being non-zero) may be divided, giving a rational number. Therefore, every element of Z×Z* is an element of some member of {Ar : r∈Q}.

    Let (m,n) ∈ Z×Z* and let q=m/n. If it is also true that p=m/n, then p=q. Therefore, if m/n ∈ Ap and m/n ∈ Aq, then p=q and Ap = Aq. This means that no element of Z×Z* is an element of two members of {Ar : r∈Q}.

    Since every element of Z×Z* is an element of exactly one member of {Ar : r∈Q}, it is a partition.

    The equivalence relation is x~y iff x and y are fractional representations of the same rational number.

  4. For r ∈ {0, 1, ..., 9} let Ar be the set of all integers with units digit (in decimal representation) equals r. Prove that {Ar : r ∈ {0, 1, ..., 9} } is a partition of Z.

    Each member of {Ar : r ∈ {0, 1, ..., 9} } contains r, so none of these sets is empty.

    Since every integer has exactly one decimal representation, with exactly one units digit, every integer is an element of exactly one member of {Ar : r ∈ {0, 1, ..., 9} }.

    Therefore, {Ar : r ∈ {0, 1, ..., 9} } partitions Z.

    The equivalence relation is p~q iff p ≅ q (mod 10).

  5. Each rational number x can be written as x = q + n/m where 0≤n/m<1. For each rational r ∈ {x : 0≤x<1} let Ar = {x ∈ Q : r is the fractional part of x}. Prove that {Ar : r ∈ {x : 0≤x<1} } partitions Q.

    Each member of {Ar : r ∈ {x : 0≤x<1} } contains r, so none of these sets is empty.

    The fractional part of a rational, x, may also be expressed as m/n = x - floor(x). Since "floor(x)" is a function, it has a single value, which makes m/n unique for each rational. Therefore every rational is in exactly one member of {Ar : r ∈ {x : 0≤x<1} }.

    Therefore, {Ar : r ∈ {x : 0≤x<1} } partitions Q.

    The equivalence relation is p~q iff (p-q) ∈ Z.

  6. For each r in R, let Ar = {(x,y) ∈ R×R : x-y=r}. Prove that {Ar : r ∈ R} partitions R×R.

    Each member of {Ar : r ∈ R} contains (r,0), so none of these sets is empty.

    x-y has a unique value, so each ordered pair (x,y) is in exactly one of these sets.

    Therefore, {Ar : r ∈ R} partitions R×R.

    The equivalence relation is (a,b)~(c,d) iff a-b = c-d.
