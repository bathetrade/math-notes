Problem I — Conjugate Subgroups

Let H < G. ∀ a ∈ G, let aHa-1 = { aha-1 : h ∈ H }. aHa-1 is called a conjugate of H. Prove the following.

  1. For each a ∈ G, aHa-1 < G.

    Let h1,h2 ∈ H. Then, ah1a-1, ah2a-1, ∈ aHa-1. Multiplying them gives: ah1a-1ah2a-1 = ah1(a-1a)h2a-1 = ah1eh2a-1 = a(h1h2)a-1. Since H is a group, (h1h2) ∈ H and a(h1h2)a-1 ∈ aHa-1. Therefore, aHa-1 is closed under multiplication.

    Let h ∈ H. Then, aha-1 ∈ aHa-1. (aha-1)-1 = (a-1)-1h-1a-1 = ah-1a-1. Since H is a group, h-1 ∈ H, which means that ah-1a-1 ∈ aHa-1, which is therefore closed under inversion.

    Since aHa-1 is a subset of G that is closed under multiplication and inversion, it is a subgroup of G.

  2. ∀ a ∈ G, H ≘ aHa-1.

    Let fa: H → aHa-1 be defined as fa(h) = aha-1.

    fa preserves products Let x,y ∈ H. Then fa(x)fa(y) = (axa-1)(aya-1) = ax(a-1a)ya-1 = axeya-1 = axya-1 = f(xy).

    fa is one-to-one Let x,y ∈ H with fa(x) = fa(y). Then:

    fa(x) fa(y) given
    axa-1 aya-1 definition of fa
    ax ay Cancellation Law
    x y Cancellation Law

    Since fa(x) = fa(y) implies x=y, fa is one-to-one.

    fa is onto Let x ∈ aHa-1. Then ∃ p ∈ H such that x = apa-1. Therefore, fa(p) = x and fa is onto aHa-1.

    Since fa is a function between groups that preserves products, it is a homomorphism. Since it is a one-to-one and onto function, it is a bijection. A bijective homomorphism is an isomorphism.

    Since fa is an isomorphism between H and aHa-1, H ≘ aHa-1. ∎

  3. H ⊲ G iff H = aHa-1 ∀ a ∈ G.

    In Chapter 14, Problem D, Exercise 5, it was shown that if H < G, H ⊲ G iff aH = Ha ∀ a ∈ G. However, the condition aH = Ha is equivalent to the condition aHa-1 = H.

    Therefore, H ⊲ G iff H = aHa-1 ∀ a ∈ G. ∎

    In the remaining exercises, let G be a finite group.

    The normalizer of H is defined as N(H) = { a ∈ G : axa-1 ∈ H ∀ x ∈ H }.

  4. If a ∈ N(H), then aHa-1 = H.

    By the definition of the normalizer, if x ∈ H, axa-1 ∈ H. Therefore, aHa-1 ⊆ H.

    As shown in Exercise 2, fa(h) = aha-1 is one-to-one. Therefore |H| ≤ |aHa-1|. The only way that a set can have cardinality less than or equal to that of a subset is if they are equipollent. A finite set can only be equipollent with a subset if they are the same set. Therefore, aHa-1 = H. ∎

  5. N(H) < G.

    Let p,q ∈ N(H). Then, ∀ x ∈ H, qxq-1 ∈ H. Therefore, pqxq-1p-1 ∈ H. Since (q-1p-1) = (pq)-1, we have (pq)x(pq)-1 ∈ H. Thus, (pq) ∈ N(H), which is therefore closed under multiplication.

    Let z ∈ N(H). Then, since zHz-1 = H, ∀ x ∈ H, ∃ y ∈ H such that x = zyz-1. But, z-1xz = z-1zyz-1z = eye = y ∈ H. Thus, z-1 ∈ N(H), which is therefore closed under inversion.

    Since N(H) is a subset of G that is closed under multiplication and inversion, N(H) < G. ∎

  6. H ⊆ N(H). Furthermore, H ⊲ N(H).

    Since H is a subgroup, it is closed under multiplication and inversion. Therefore, ∀ a,x ∈ H, axa-1 ∈ H. Thus, a ∈ H implies that axa-1 ∈ H ∀ x ∈ H. This means that ∀ a ∈ H, a ∈ N(H), making H ⊆ N(H).

    Since, by definition, ∀ a ∈ N(H), aha-1 ∈ H ∀ h ∈ H, H ⊲ N(H).

    For the remaining exercises, let N = N(H).
