Problem C — Elementary Properties of Order

Let a, b, and c be elements of the group G. Prove the following:

  1. Ord(a) = 1 if and only if a=e.

    If a=e, then a1 = e1 = e. Therefore, ord(a) = 1.

    If ord(a) = 1, then a1 = e. Since a1 = a, a=e.

  2. If ord(a) = n, then an-r = (ar)-1.

    an-r an+(-r) Definition of subtraction
    ana-r ax+y = axay
    (e)a-r ord(a) = n
    a-r identity

  3. If ak = e where k is odd, then ord(a) is odd.

    Since k is odd, it is not a multiple of 2. Therefore, by Theorem 5, ord(a) is not a multiple of 2.

  4. Ord(a) = ord(bab-1).

    As shown in Chapter 4, Problem H, (bab-1)n = banb-1.

    If ord(a) = p, then (bab-1)p = bapb-1 = beb-1 = bb-1 = e. Since ord(bab-1) is the least positive integer with this property, ord(bab-1) ≤ p = ord(a).

    Lemma If xyx-1 = e, then y = e.

    xyx-1 e given
    xyx-1x ex post-multiply by x
    xye ex inverse
    xy x identity
    x-1xy x-1x premultiply by x-1
    ey e inverse
    y e identity

    If ord(bab-1) = q, then (bab-1)q = e. Since e = (bab-1)q = baqb-1, by the lemma aq = e. Since ord(a) is the least positive integer with this property, ord(a) ≤ q = ord(bab-1).

    Since ord(bab-1) ≤ ord(a) and ord(a) ≤ ord(bab-1), ord(a) = ord(bab-1).

  5. Ord(a) = ord(a-1).

    Let ord(a) = n.

    e aa-1 inverse
    (aa-1)n identity
    an(a-1)n an element commutes with its inverse
    e(a-1)n ord(a) = n
    (a-1)n identity

    Since (a-1)n = e, ord(a) = n is a multiple of ord(a-1). A similar argument shows that ord(a-1) is a multiple of ord(a). Therefore, ord(a-1) = ord(a).

  6. ord(ab) = ord(ba).

    Let ord(ab) = p. Then e = (ab)p = (ab)p-1ab. This means that (ab)p-1a = b-1, and they commute. This gives e = b(ab)p-1a = (ba)p.

    Therefore, ord(ba) is a multiple of p = ord(ab). A similar argument shows that ord(ab) is a multiple of ord(ba), which means that ord(ab) = ord(ba).

  7. Ord(abc) = ord(cab) = ord(bca).

    Let ord(abc) = q. Then e = (abc)q = (abc)q-1(abc) = (abc)q-1(ab)c. Therefore, (abc)q-1(ab) = c-1, and they commute, giving e = c(abc)q-1(ab) = (cab)q. This means that ord(cab) is a multiple of q = ord(abc).

    Let ord(cab) = r. Then e = (cab)r = (cab)r-1(cab) = (cab)r-1(ca)b. Therefore, (cab)r-1(ca) = b-1, and they commute, giving e = b(cab)r-1(ca) = (bca)r. This means that ord(bca) is a multiple of r = ord(cab).

    Let ord(bca) = s. Then e = (bca)s = (bca)s-1(bca) = (bca)s-1(bc)a. Therefore, (bca)s-1(bc) = a-1, and they commute, giving e = a(bca)s-1(bc) = (abc)s. This means that ord(abc) is a multiple of s = ord(bca).

    Since ord(cab) is a multiple of ord(abc), ord(bca) is a multiple of ord(cab), and ord(abc) is a multiple of ord(bca), it must be true that ord(abc) = ord(cab) = ord(bca).

  8. Let x = a1a2... an and let y = ak... ana1... ak-1 be a product of the same factors as x, permuted cyclically. Prove that ord(x) = ord(y).

    Let ord(x) = p. Then e = xp = (a1a2... an)p = (a1a2... an)p-1 (a1a2... an) = (a1a2... an)p-1 (a1a2... ak-1) (ak... an). Therefore, (a1a2... an)p-1 (a1a2... ak-1) = (ak... an)-1, and they commute, giving e = (ak... an). (a1a2... an)p-1 (a1a2... ak-1) = (ak... ana1... ak-1)p. Therefore, ord(y) is a multiple of p = ord(x).

    A similar argument shows that ord(x) is a multiple of ord(y), which means that ord(x) = ord(y).
