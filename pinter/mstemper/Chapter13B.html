Problem B — Examples of Cosets in Infinite Groups

Describe the cosets of the subgroups described in 1-5:

  1. The subgroup ⟨3⟩ of Z.

    For i ∈ {0, 1, 2}, let Ci = {p ≅ i (mod 3) : p ∈ Z}. Then the set of cosets is C = {C0, C1, Ci}.

  2. The subgroup Z of R.

    For r ∈ R and 0≤r<1, let Cr = {x ∈ R : x - floor(x) = r}. Then the set of cosets is C = {Cr : r ∈ R ∧ 0≤r<1}.

  3. The subgroup H = {2n : n ∈ Z} of R*.

    For r ∈ R and 1≤r<2, let Cr = {r·2n : n ∈ Z}. Then the set of cosets is C = {Cr : r ∈ R ∧ 1≤r<2}.

    • The subgroup ⟨1/2⟩ of R*.

      For r ∈ R and 1≤r<2, let Cr = {r·2n : n ∈ Z}. Then the set of cosets is C = {Cr : r ∈ R ∧ 1≤r<2}.

      In other words, the same as in the previous exercise.

    • The subgroup ⟨1/2⟩ of R.

      For r ∈ R and 0≤r<1/2, let Cr = {x ∈ R : (2x - floor(2x))/2 = r}. Then the set of cosets is C = {Cr : r ∈ R ∧ 0≤r<1/2}.

  5. The subgroup H = {(x,y) : x=y} of R × R.

    For r ∈ R, let Cr = {(x,y) ∈ R × R : x-y = r}. Then the set of cosets is C = {Cr : r ∈ R ∧ 0≤r<1/2}.

  6. For any positive integer, n, what is the index of ⟨n⟩ in Z?

    (Z : ⟨n⟩) = n.

  7. Find a subgroup of R* which has an index equal to 2.

    R+ is such a subgroup.
