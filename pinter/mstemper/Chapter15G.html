Problem G — Using the Class Equation to Determine the Size of the Center

Let G be a finite group. Elements a and b ∈ G are called conjugates of one another (a~b) iff a = xbx-1 for some x ∈ G.

Let |G| = pk, where p is prime. Let C be the center of G. Let S1, … St be the conjugacy classes of G, and k1, … kt be thier sizes.

  1. Prove that the conjugacy class of a contains a (and no other element) iff a ∈ C.

    If Let a ∈ C. If a~b, then a = xbx-1 for some x ∈ G. This is equivalent to b = x-1ax. Since a ∈ C, a commutes with x, giving b = x-1xa = ea = a. Therefore, if a ∈ C, a~b ⇒ a=b.

    Only If If a ∉ C, then ∃ y ∈ G such that ay ≠ ya. Let b = y-1ay. Then, yb = ay ≠ ya. Since yb ≠ ya, b ≠ a, by Cancellation. From the definition of b, a = yby-1, giving a~b, where a≠b. Therefore, if a ∉ C, ∃ b such that a~b and a≠b.

    ∎

  2. Let |C| = c. Prove that |G| = c + ks + … + kt, where ks, …, kt are the sizes of the distinct conjugacy classes of elements not in C.

    Since ~ is an equivalence relation on G, it partitions G. Since every element of G is in exactly one class of the partition (conjugacy classes), |G| = k1 + … + kt, where the ki are the sizes of all of the classes.

    The conjugacy classes of the elements of C are all singletons, as shown in Exercise 1. Therefore, there are c of them. If we remove the sizes of those c singletons from the sum, and replace them with c, the total is unchanged, since |G| - c·1 + c = |G|. ∎

  3. Prove that ∀ i ∈ {s, s+1, …, t}, ki is a power of p.

    The size, ki, of conjugacy class i is equal to the number of distinct conjugates of any element of class i. In Chapter 13, Problem I, Exercise 6, it was shown that if a ∈ G, the number of distinct conjugates of a is (G : Ca), where Ca is the centralizer of a.

    The important fact is that (G : Ca) | G. Therefore, ki | G. Since |G| is a power of a prime, ki must be a power of that prime (not a larger power).

  4. Solving |G| = c + ks + … + kt for c, explain why c must be a multiple of p.

    |G| - ks - … - kt = c. Since all of the terms on the left are multiples of p (powers, actually, but that's more than we need), this may be rewritten as gp - xsp - … - xtp = c, or (g -xs - … - xt)p = c.

    Therefore, c is a multiple of p.

  5. Prove that if |G| = p2, G is Abelian.

    The previous exercise showed that the order of the center of G must be a multiple of p. By Lagrange's Theorem, it must divide p2. Since p is prime, the only multiples of p that divide p2 are p or p2.

    |C| = p In this case, |G/C| = |G| / |C| = p. Since |G/C| is prime, G/C is cyclic.

    By Exercise F of this chapter, if G/C is cyclic, then G is Abelian.

    |C| = p2 In this case |C| = |G|, which makes the center the entire group, which is therefore Abelian.

  6. Prove that if |G| = p2, either G ≘ Zp2 or G ≘ Zp × Zp.

    If G ≘ Zp2, we are done.

    Assume that G ≇ Zp2. Then, no element of G has order p2. By Lagrange's Theorem, every element must have order 1 or order p, since p is prime. Therefore, aside from the identity, every element must have order p.

    Let a ∈ G-{e}. |⟨a⟩| = p. Let b ∈ G-⟨a⟩. Again, |⟨b⟩| = p.

    Then, G = ⟨a,b⟩.
