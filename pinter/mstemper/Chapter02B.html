Problem B — Properties of Operations

For each of the following operations (on R), determine whether it is commutative, associative, has an identity element, has an inverse.

  1. x*y = x + 2y + 4

    It is not commutative, since (x + 2y + 4) does not generally equal (y + 2x + 4).

    (x*y)*z = (x + 2y + 4)*z = x + 2y + 4 + 2z + 4 = x + 2y + 2z + 8.
    x*(y*z) = x*(y + 2z + 4) = x + 2y + 4z + 8 + 4 = x + 2y + 4z +12.
    Since these are not generally equal, this is not associative.

    Solving e*y = y for e gives: e + 2y + 4 = y; e = -(y+4) as a left identity. Since this is a function of y, this operation does not have a identity.

    Since the operation does not have an identity, it is not meaningful to ask if it has an inverse.

  2. x*y = x + 2y - xy

    It is not commutative, since (x + 2y - xy) does not generally equal (y + 2x - yx).

    (x*y)*z = (x + 2y - xy)*z = (x + 2y - xy + 2z - z(x + 2y - xy)) = (x + 2y - xy + 2z - xz - 2yz + xyz).
    x*(y*z) = x*(y + 2z - yz) = (x + 2(y + 2z - yz) - x(y + 2z - yz)) = (x + 2y + 4z - 2yz - xy - 2xz + xyz).
    Since these are not generally equal, this is not associative.

    Solving e*y = y for e gives: e + 2y - ey = y; e = -y/(1-y). Since this is a function of y, this operation does not have a identity.

    Since the operation does not have an identity, it is not meaningful to ask if it has an inverse.

  3. x*y = |x + y|

    Since x+y = y+x, |x + y| = |y + x|, so this is commutative.

    Let x=1, y=-2, and z=3.
    Then (x*y)*z = ||1 + -2| + 3| = ||-1| + 3| = |1 + 3| = 4.
    x*(y*z) = |1 + |-2 + 3|| = |1 + |1|| = |1 + 1| = 2.
    This operation is not associative.

    Since the operation always returns a non-negative number, it can not have an identity element.

    Since the operation does not have an identity, it is not meaningful to ask if it has an inverse.

  4. x*y = |x - y|

    Since (x-y) = -(y-x), |x - y| = |y - x|, and this is commutative.

    Let x=1, y=2, and z=3.
    Then (x*y)*z = ||1 - 2| - 3| = ||-1| - 3| = |1 - 3| = 2.
    x*(y*z) = |1 - |2 - 3|| = |1 - |-1|| = |1 - 1| = 0.
    This operation is not associative.

    Since the operation always returns a non-negative number, it can not have an identity element.

    Since the operation does not have an identity, it is not meaningful to ask if it has an inverse.

  5. x*y = xy + 1

    Since xy=yx, xy + 1 = yx + 1, and this is commutative.

    (x*y)*z = (xy + 1)*z = xyz + z + 1.
    x*(y*z) = x*(yz + 1) = xyz + x + 1.
    Since these are not generally equal, this is not associative.

    Solving e*y=y for e gives: ey + 1 = y; e = 1 - 1/y. Since this is a function of y, this operation does not have a identity.

    Since the operation does not have an identity, it is not meaningful to ask if it has an inverse.

  6. x*y = max(x,y)

    Since the maximum of two numbers is not affected by their order, this is a commutative operation.

    Since the maximum of three numbers is not affected by their order, this is an associative operation.

    Let e be a proposed identity element for this operation. Let y = e-1. Then, max(e,y) = e rather than y, so e is not an identity element.

    Since the operation does not have an identity, it is not meaningful to ask if it has an inverse.

  7. x*y = (xy)/(x+y+1), on R+

    Since xy=yx and (x+y+1)=(y+x+1), this is commutative.

    (x*y)*z = (xy/(x+y+1))*z = xyz/(xy+xz+yz+x+y+z+1)
    x*(y*z) = x*((yz)/(y+z+1)) = xyz/(xy+xz+yz+x+y+z+1)
    Surprisingly (to me), this operation is associative.

    Solving e*y=y for e gives ey/(e+y+1)=y → e/(e+y+1)=1 → e=e+y+1 → y=-1. Since e drops out, this operation has no identity element.

    Since the operation does not have an identity, it is not meaningful to ask if it has an inverse.
