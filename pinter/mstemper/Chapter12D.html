Problem D — Equivalence Relations on Groups

Let G be a group. In each of the following, a relation on G is defined. Prove it is an equivalence relation. Then describe the equivalence class of e.

  1. If H is a subgroup of G, a~b iff ab-1 ∈ H.

    Reflexive Since a subgroup contains e, aa-1 ∈ H, and this relation is reflexive.

    Symmetric Let a~b. Then, ab-1 = h ∈ H, which gives:

    ab-1 h given
    (ab-1)-1 h-1 invert both sides
    (b-1)-1a-1 h-1 inverse
    ba-1 h-1 inverse of the inverse

    If a~b, then ab-1 = h ∈ H, which gives ba-1 = h-1. Since H is a subgroup, which is closed under inversion, h-1 ∈ H. Therefore, ba-1 ∈ H, making the relation symmetric.

    Transitive Let a~b and b~c. Then, ab-1 ∈ H and bc-1 ∈ H. Since subgroups are closed under multiplication, it must be true that (ab-1)(bc-1) ∈ H.

    (ab-1)(bc-1) a(b-1b)c-1 associativity
    aec-1 inverse
    ac-1 identity

    Therefore, ac-1 ∈ H, which means that a~c. Since a~b and b~c implies a~c, the relation is transitive.

    Since the relation is reflexive, symmetric, and transitive, it is an equivalence relation.

    Since e = e-1, if h ∈ H, then he-1 = he = h ∈ H. Therefore, all members of H are in the equivalence class of e. By the same reasoning, if p ∉ H, p cannot be in the equivalence class of e. Therefore, the equivalence class of e is H.

  2. If H is a subgroup of G, a~b iff a-1b ∈ H. Is this the same equivalence relation as in part 1? Prove or disprove.

    Reflexive Since a subgroup contains e, a-1a ∈ H, and this relation is reflexive.

    Symmetric Let a~b. Then, a-1b = h ∈ H, which gives:

    a-1b h given
    (a-1b)-1 h-1 invert both sides
    b-1(a-1)-1 h-1 inverse
    b-1a h-1 inverse of the inverse

    If a~b, then a-1b = h ∈ H, which gives b-1a = h-1. Since H is a subgroup, which is closed under inversion, h-1 ∈ H. Therefore, b-1a ∈ H, making the relation symmetric.

    Transitive Let a~b and b~c. Then, a-1b ∈ H and b-1c ∈ H. Since H is a subgroup, it is closed under multiplication, so it must be true that (a-1b)(b-1c) ∈ H. Since (a-1b)(b-1c) = a-1c, a~c. Since a~b and b~c together imply a~c, the relation is transitive.

    Since this relation is reflexive, symmetric, and transitive, it is an equivalence relation.

    Since e = e-1, if h ∈ H, then e-1h = eh = h ∈ H. Therefore, all members of H are in the equivalence class of e. By the same reasoning, if p ∉ H, p cannot be in the equivalence class of e. Therefore, the equivalence class of e is H.

    Call the relation from part 1 "R1" and this one "R2". Let G = {I, A, B, C, D, K} be the six-matrix group from pages 28 and 29 of the text, and let H = {I, A}.

    The equivalence classes of R1 are {{I,A}, {B,C}, {D,K}}, while those of R2 are {{I,A}, {B,K}, {C,D}}.

    Therefore, R1 and R2 are not the same equivalence relation.

  3. Let a~b iff ∃ x ∈ G such that a = xbx-1.

    Reflexive Since a = eae-1, a~a and this relation is reflexive.

    Symmetric If a~b then a = xbx-1 for some x ∈ G. If x ∈ G, then x-1 ∈ G, and x-1ax = b. Therefore a~b implies b~a, making the relation symmetric.

    Transitive Let a~b and b~c. Then, ∃ x,y ∈ G such that a = xbx-1 and b = ycy-1. Therefore, a = xycy-1x-1. Since xy ∈ G and y-1x-1 = (xy)-1, a~c. Since a~b and b~c together imply a~c, this relation is transitive.

    Since this relation is reflexive, symmetric, and transitive, it is an equivalence relation.

    The equivalence class of e is {e}. Let e~b. Then, by symmetry, b~e and b = xex-1 = xx-1 = e.

  4. Let a~b iff ∃ k ∈ Z such that ak = bk.

    Reflexive Since ak = ak, a~a and this relation is reflexive.

    Symmetric If a~b then ak = bk for some k ∈ Z. Therefore, bk = ak, which means b~a. Since a~b implies b~a, this relation is symmetric.

    Transitive If a~b and b~c, then ∃ j,k ∈ Z such that aj = bj and bk = ck. Then, ajk = cjk and a~c. Since a~b and b~c together imply a~c, this relation is transitive.

    Since this relation is reflexive, symmetric, and transitive, it is an equivalence relation.

    The equivalence class of e is {x ∈ G : ord(x) is finite}.

  5. Let a~b iff ab-1 commutes with every x ∈ G.

    Reflexive Since aa-1 = e, which commutes with every element in a group, a~a and this relation is reflexive.

    Symmetric Assume a~b. Then:

    ab-1x xab-1 a~b
    ab-1xb xa post-multiply both sides by b
    ab-1xba-1 x post-multiply both sides by a-1
    xba-1 (ab-1)-1x pre-multiply both sides by (ab-1)-1
    xba-1 ba-1x inverse of ab-1

    Therefore, if ab-1 commutes with x, ba-1 does, which means that a~b implies b~a, and this relation is symmetric.

    Transitive If b~c, then for any y ∈ G, bc-1y = ybc-1. Let x = bc-1y = ybc-1. Then, since a~b, ab-1x = xab-1. Substituting x = bc-1y on the left-hand side and x = ybc-1 gives ab-1bc-1y = ybc-1ab-1. Because b~c, bc-1 commutes with ab-1, giving ab-1bc-1y = yab-1bc-1. Since b-1b = e, both sides may be simplified, leaving ac-1y = yac-1. This means that taking a~b and b~c together implies a~c, and this relation is transitive.

    Since this relation is reflexive, symmetric, and transitive, it is an equivalence relation.

    The equivalence class of e is the set of all elements that commute with every element of G. (Note that symmetry shows that if an element commutes, its inverse will, as well. In fact, this is a subgroup of G.)

  6. Let a~b iff ab-1 = ck for some fixed c ∈ G.

    Reflexive Since aa-1 = e = c0, a~a and this relation is reflexive.

    Symmetric If a~b then ab-1 = ck. Then, c-k = ba-1. Since this is a power of c, a~b implies b~a and this relation is symmetric.

    It would have been nice if Pinter had picked something other than "c" to define the relation.

    Transitive If a~b then ab-1 = cj. If b~d then bd-1 = ck. Multiplying them gives ab-1bd-1 = cj+k, or ad-1 = cj+k, or a~d. Since a~b and b~d taken together imply a~d, this relation is transitive.

    Since this relation is reflexive, symmetric, and transitive, it is an equivalence relation.

    The equivalence class of e is the set of powers of c, which is a cyclic subgroup of G.
