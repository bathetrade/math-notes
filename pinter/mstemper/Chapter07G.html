Problem G — Symmetries of Polynomials

List the symmetries of each of the following polynomials, and give the group table:

  1. p =  x1x2 + x2x3

    The symmetries:

    P0 = 
    1 2 3
    1 2 3
    P1 = 
    1 2 3
    3 2 1

    The table:

    ∘ P0 P1
    P0 P0 P1
    P1 P1 P0

  2. p =  (x1-x2)(x2-x3)(x1-x3)

    The symmetries:

    P0 = 
    1 2 3
    1 2 3
    P1 = 
    1 2 3
    2 3 1
    P2 = 
    1 2 3
    3 1 2

    The table:

    ∘ P0 P1 P2
    P0 P0 P1 P2
    P1 P1 P2 P0
    P2 P2 P0 P1

  3. p =  x1x2 + x2x3 + x1x3

    The symmetries:

    P0 = 
    1 2 3
    1 2 3
    P1 = 
    1 2 3
    2 3 1
    P2 = 
    1 2 3
    3 1 2
    P3 = 
    1 2 3
    2 1 3
    P4 = 
    1 2 3
    1 3 2
    P5 = 
    1 2 3
    3 2 1

    The table:

    ∘ P0 P1 P2 P3 P4 P5
    P0 P0 P1 P2 P3 P4 P5
    P1 P1 P2 P0 P5 P3 P4
    P2 P2 P0 P1 P4 P5 P3
    P3 P3 P4 P5 P3 P1 P2
    P4 P4 P5 P3 P2 P0 P1
    P5 P5 P3 P4 P1 P2 P0

  4. p =  (x1-x2)(x3-x4)

    The symmetries:

    P0 = 
    1 2 3 4
    1 2 3 4
    P1 = 
    1 2 3 4
    3 4 1 2

    The table:

    ∘ P0 P1
    P0 P0 P1
    P1 P1 P0
