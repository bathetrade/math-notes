Problem K — Coding Theory: Coset Decoding

This problem relies heavily on Chapter 3, Problem F and Problem G, and Chapter 5, Problem H.

  1. Let C1 be the code described in Chapter 3, Problem G, Exercise 1.

    1. List the elements in each of the cosets of C1.

      • C1 + 00000 = {00000, 00111, 01001, 01110, 10011, 10100, 11010, 11101} = C1
      • C1 + 00001 = {00001, 00110, 01000, 01111, 10010, 10101, 11011, 11100}
      • C1 + 00010 = {00010, 00101, 01011, 01100, 10001, 10110, 11000, 11111}
      • C1 + 00100 = {00100, 00011, 01101, 01010, 10111, 10000, 11110, 11001}

    2. Find a coset leader of each coset.

      • C1 + 00000 ⇒ 00000
      • C1 + 00001 ⇒ 00001 or 01000
      • C1 + 00010 ⇒ 00010
      • C1 + 00100 ⇒ 00100 or 10000

    3. Use the procedure described in the text to decode the following words:

      • 11100:

        11100 ∈ C1 + 00001. Its coset leader is 00001. 11100 + 00001 = 11101.

        Since C1 + 00001 has two words of equal (minimum) weight, I'll also try 01000. 11100 + 01000 = 10100.

        Both of these words are in C1, so the ambiguity does not allow resolution.

      • 01101:

        01101 ∈ C1 + 00100. Its coset leader is 00100. 01101 + 00100 = 01001.

        Since C1 + 00100 has two words of equal (minimum) weight, I'll also try 10000. 01101 + 10000 = 11101.

        Again, both of these words are in C1.

      • 11011:

        11011 ∈ C1 + 00001. 11011 + 00001 = 11010.

        (I'm not going to continue trying both potential coset leaders.)

      • 00011:

        00011 ∈ C1 + 00100. 00011 + 00100 = 00111.

  3. Let C be a code and let H be the parity-check matrix of C. Prove that x and y are in the same coset of C iff Hx = Hy.

    If Assume that Hx = Hy. By Exercise 8 of Chapter 5, Problem H, this implies that H(x+y) = 0.

    If Ha = 0, then a ∈ C, by the definition of the parity-check matrix. Therefore, x+y ∈ C.

    Let x+y = b ∈ C. Then x = b+y (since all elements are self-inverses). Therefore, x ∈ C + y, which is a coset of C. Since 0 ∈ C, y = 0+y ∈ C + y.

    Therefore, Hx = Hy implies that x and y are in the same coset of C.

    Only If Assume that x, y ∈ C + d.

    Then, ∃ p ∈ C such that x = p + d. Also, ∃ q ∈ C such that y = q + d.

    Since all elements are self-inverses, we have x + d = p. Since p ∈ C, H(x + d) = 0. Therefore (again by Exercise 8 of Chapter 5, Problem H), Hx = Hd.

    A similar argument shows that Hy = Hd.

    Therefore, x, y ∈ C + d implies Hx = Hy.

  4. Let a code C have q cosets, and let the coset leaders be e1, e2, ... eq. Explain why the following is a valid method of decoding a received word x:

    The syndrome of q, syn(q) = Hq.

    Compare syn(x) with syn(e1), syn(e2), ... syn(eq) and find the coset leader ei such that syn(x) = syn(ei). Then x is to be decoded to x + ei.

    As shown in Exercise 3, if syn(x) = syn(ei), then x is in the same coset as ei.

    Since x is in the same coset as ei, and ei is the coset leader, decoding x to x + ei is simply the method described in the text.
