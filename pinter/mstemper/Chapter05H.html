  3. With the weight of a word, w(a), defined as the number of one bits, and the distance between two words, d(a,b) as in Problem G, prove that d(x,y) = w(x+y).

    By the definition of addition, (x+y) has a one bit only in the positions where xi≠yi, so the weight of the sum is the number of positions where they differ. This is the definition of their distance, so d(x,y) = w(x+y).

  4. Prove that d(x,0) = w(x).

    The bits of x and 0 differ only where xi=1, so d(x,0) is the number of one bits in x, which is the definition of w(x).

  5. Prove that the minimum distance of a group code C is equal to the minimum weight of C.

    This only works for groups with order of at least three. But, there is only one distance defined for smaller groups, and that is the weight of the non-zero element (by the previous exercise).

    Let m be the minimum distance of C. Then, there are a,b (a≠b and a≠0≠b) such that d(a,b)=m. Since C is a group, (a+b)∈C. By the previous exercise, w(a+b)=m.

    Let x∈C such that x≠0. By the laws of groups, for any c∈C (x≠c≠0), ∃d∈C (x≠d≠0) such that c+d=x. Then, w(x)=w(c+d)=d(c,d)≥m. So, any (non-zero) element must have a weight at least that of the minimum distance, and the minimum distance is therefore the minimum weight.

  6. Prove that ...

    1. If x and y have even weight, so does x+y.

      Let k be the number of positions where both x and y have a one. These positions will have a zero in x+y. So, w(x+y) = (w(x)-k) + (w(y)-k) = (w(x)+w(y)) - 2k. Since w(x) and w(y) are even, this is also even.

    2. If x and y have odd weight, x+y has even weight.

      Let k be the number of positions where both x and y have a one. These positions will have a zero in x+y. So, w(x+y) = (w(x)-k) + (w(y)-k) = (w(x)+w(y)) - 2k. Since w(x) and w(y) are odd, this is even.

    3. If x has odd weight and y has even weight, x+y has odd weight.

      As before, w(x+y) = (w(x)+w(y)) - 2k. Since w(x) is odd and w(y) is even, this is odd.

  7. Prove that in any group code, either all of the words have even weight or half have even weight and half have odd weight.

    For a group code C, assume that there are n>0 elements with odd weight. Let O be the set of all elements with odd weight. Select one element x∈O and form the sums E={x+y | y∈O}. By Exercise 6b, all elements of E have even weight. In a group, x+a=x+b iff a=b, so E contains n distinct elements. The set P={x+y | y∈E} only contains elements with odd weight, by Exercise 6c, so P=O, and C = O∪E. Since O and E are disjoint sets, |C|=2n, and half the elements have odd weight and half have even weight.

    If there are not n>0 elements with odd weight, then all of the elements have even weight.
