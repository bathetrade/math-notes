Problem D — Further Elementary Consequences of Lagrange's Theorem

Let G be a finite group, and let H and K be subgroups of G. Prove the following:

  1. Suppose H ⊆ K. Then, (G : H) = (G : K)·(K : H).

    As shown at the very end of the chapter, |G| = (G : H)·|H| and |H| = (H : K)·|K|.

    Substituting the second into the first gives: |G| = (G : H)·(H : K)·|K|.

    Dividing both sides by |K| gives: |G|/|K| = (G : H)·(H : K).

    Since |G|/|K| = (G : K), we have (G : K) = (G : H)·(H : K).

  2. The order of H ∩ K is a common divisor of both |H| and |K|.

    Since H ∩ K is a group, it is a subgroup of both H and K. Therefore, by Lagrange's Theorem, |H ∩ K| divides both |H| and |K|.

  3. Let H have order m and K have order n, where m and n are relatively prime. Then H ∩ K = {e}.

    As shown in the previous exercise, |H ∩ K| divides both |H| and |K|. Since m and n are relatively prime, gcd(m,n) = 1. Therefore, |H ∩ K| = 1, and H ∩ K = {e}.

  4. Suppose H ≠ K, and they both have the same order, p, which is prime. Then H ∩ K = {e}.

    Let x ∈ H ∩ K. Since |H| = p, which is prime, ord(x) = p unless x=e. Therefore, if x≠e, H = ⟨x⟩. Since Since |K| = p, which is prime, ord(x) = p unless x=e. Therefore, if x≠e, K = ⟨x⟩.

    Therefore, if x≠e then H = K. Since it was stated that this is not true, we must have x=e as the only element of the intersection.

  5. Suppose H has index p and K has index q, where p and q are distinct primes. Then the index of H ∩ K is a multiple of pq.

    H ∩ K ⊂ H. By Exercise 1, (G : H ∩ K) = (G : H)·(H : H ∩ K) = p·(H : H ∩ K).

    Similarly, H ∩ K ⊂ K, which means that (G : H ∩ K) = (G : K)·(K : H ∩ K) = q·(K : H ∩ K).

    Therefore, (G : H ∩ K) is a multiple of both p and q, or a multiple of lcm(p,q). Since p and q are distinct primes, lcm(p,q) = pq, so it must be a multiple of their product.

  6. If G is an Abelian group of order n, and m and n are relatively prime, then f(x) = xm is an automorphism of G.

    (f(x-1))m = x-1·m = xm·-1 = (xm)-1 = (f(x))-1, so f preserves inverses.

    Since G is Abelian, f(xy) = (xy)m = xmym = f(x)f(y), so f preserves products.

    Let y ∈ G, y≠e, and let ord(y) = p. Since y≠e, p > 1. Since ⟨y⟩ is a subgroup of G, |⟨y⟩| = ord(y) = p, and p|n. Since gcd(m,n) = 1, p does not divide m, so ym≠e.

    Assume that for some a,b ∈ G, a,b ≠ e, f(a) = f(b). Since b ∈ G, b = ax for some x ∈ G, so f(a) = f(ax), or am = (ax)m = amxm. Pre-multiplying both sides by a-m gives e = xm. As shown above, if x≠e then xm≠e. Therefore, x=e, giving a=b.

    Therefore, f(a) = f(b) implies a=b. This means that G is one-to-one. Any one-to-one function between finite sets with the same cardinality is a bijection, so f is an automorphism on G.
