Problem G — Relationship Between ord(a) and ord(ak)

Let a be a element of order n in a group G. Use elementary results from number theory.

  1. Prove that, if m and n are relatively prime, ord(am) = n.

    Let ord(am) = k. Then, e = (am)k = amk.

    By Theorem 5, n | mk. Since m and n are relatively prime, n | k, and there is a q (1≤q) such that k = nq.

    (am)k = (am)nq = amnq = (an)mq = emq = e is true regardless of the value of q, which means that it is true for q=1. This gives us ord(am) = k = nq = n.

  2. Prove that if ord(am) = n, then m and n are relatively prime.

    Let gcd(m,n) = q. Then we can write m = m'q and n = n'q. (am)n' = (am'q)n' = am'qn' = (aqn')m' = (an)m' = em' = e.

    Since ord(am) = n, n is the smallest positive integer for which (am)n = e. Since (am)n' = e, n' is a multiple of n, by Theorem 5.

    The gcd of two integers is a positive integer, by definition. Given the assumption that n = qn', the only possible value for q is q=1, giving n = n'.

    Therefore, gcd(m,n) = 1. In other words, that m and n are relatively prime.

  3. Let l = lcm(m,n) and let l/m = k. Explain why (am)k = e.

    (am)k = (am)l/m = (am)lcm(m,n)/m = am lcm(m,n)/m = alcm(m,n) = e, since lcm(m,n) is, by definition, a multiple of n and ord(a) = n.

  4. Prove that if (am)t = e, then n | mt.

    e = (am)t = amt. Since n = ord(a), by Theorem 5, n | mt. Therefore, mt is a multiple of both m and n.

    This means that lcm(m,n) ≤ mt.

  5. Use the results of the two previous exercises to prove that ord(am) = lcm(m,n)/m.

    By Exercise 3, (am)lcm(m,n)/m = e.

    By Exercise 4, if (am)t = e, then lcm(m,n)/m ≤ t.

    Combining these results shows that lcm(m,n)/m = ord(am).
