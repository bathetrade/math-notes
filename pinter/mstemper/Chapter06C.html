Problem C — Functions on Arbitrary Sets and Groups

Determine whether each of the following functions is injective or surjective.

  1. f:A×B→A, defined by f(x,y)=x.

    If |B|>1, this function is not injective. Let v,w∈B with v≠w. Then, for any x∈A, f(x,v)=x=f(x,w).

    This function is surjective. Let a∈A and b∈B. Then, f(a,b)=a.

  2. f:A×B→B×A, defined by f(x,y)=(y,x).

    This function is injective. Let f(p,q)=f(r,s). Then, (q,p)=f(p,q)=f(r,s)=(s,r). By the definition of ordered pairs, this gives us q=s and p=r, which means that (p,q)=(r,s).

    This function is surjective. Let (y,x) be any element of B×A. Then f(x,y)=(y,x).

  3. f:A→A×B, defined by f(x)=(x,b), for some fixed b∈B.

    This function is injective. Let f(p)=f(q). Then, (p,b)=(q,b), and by the definition of ordered pairs, p=q.

    If |B|>1, this function is not surjective. Let c∈B, such that c≠b. Then, for any a∈A, f(a)≠(a,c).

  4. G is a group, with a∈G, and f:G→G is defined as f(x)=ax.

    This function is injective. Let f(p)=f(q).

    f(p) f(q) given
    ap aq definition of f
    a-1ap a-1aq pre-multiply by a-1
    ep eq inverse
    p q identity

    This function is surjective. Let v∈G. Then, f(a-1v)=aa-1v=ev=v.

  5. G is a group, and f:G→G is defined as f(x)=x-1.

    Since the inverse of an element is unique, this is an injection.

    Since every element of a group has an inverse, this is a surjection.

  6. G is a group, and f:G→G is defined as f(x)=x2.

    Whether this is injective or not depends upon the group structure. For instance, it is injective in the three-element group and the additive group of the integers. However, it is not injective in the dihedral group D4, or the multiplicative group of the non-zero reals.

    Again, whether this is surjective or not depends upon the group structure. It is surjective in the cyclic group of order five and the multiplicative group of the non-zero reals. It is not surjective in the dihedral group D4, or the additive group of the integers.
