Problem J — Group Acting on a Set

Let A be a set and let G be any subgroup of SA. G is a group of permutations of A; we say it is a group acting on the set A. Assume here that G is a finite group. If u ∈ A, the orbit of u (with respect to G) is the set O(u) = {g(u) : g ∈ G}.

If u ∈ A, the stabilizer of u is the set Gu = {g ∈ G : g(u) = u}.

  1. Define a relation ~ on A by u~v iff g(u) = v for some g ∈ G. Prove that ~ is an equivalence relation on A and that the orbits are its equivalence classes.

    Reflexive Since G is a subgroup, it contains the identity permutation, (). Thus, a~a and this relation is reflexive.

    Symmetric Let a~b. Then, for some g ∈ G, g(a) = b. Since G is a subgroup, g-1 ∈ G. g-1(b) = a. Thus, b~a and this relation is symmetric.

    Transitive Let a~b and b~c. Then, for some g,h ∈ G, g(a) = b and h(b) = c. Since G is a subgroup, it is closed under composition, h∘g ∈ G. (h∘g)(a) = h(g(a)) = c. Thus, this relation is transitive.

    Since this relation is reflexive, symmetric, and transitive, it is an equivalence relation.

    Since two elements are in the same orbit iff they are equivalent under this relation, the orbits are the equivalence classes.

  2. Prove that Gu is a subgroup of G.

    Let h ∈ Gu. Since G is a group, h-1 ∈ G. Since h(u) = u, it must also be true that h-1(u) = u. Therefore, h-1 ∈ Gu, and Gu is closed under inversion.

    Let p,q ∈ Gu. Then, (p∘q)(u) = p(q(u)) = p(u) = u. Therefore, p∘q ∈ Gu, and Gu is closed under multiplication.

    Since Gu is closed under inversion and multiplication, it is a subgroup.

  3. Let α = (12)(34)(56) and β = (23) in S6. Let G = {ε, α, β, αβ, βα, αβα, βαβ, (αβ)2} < S6. Find the following:

    • O(1):

      ε(1) = 1, α(1) = 2, β(1) = 1, αβ(1) = 2, βα(1) = 3, αβα(1) = 4, βαβ(1) = 3, (αβ)2(1) = 4. Therefore, O(1) = {1, 2, 3, 4}.

    • O(2):

      ε(2) = 2, α(2) = 1, β(2) = 3, αβ(2) = 4, βα(2) = 1, αβα(2) = 2, βαβ(2) = 4, (αβ)2(2) = 3. Therefore, O(2) = {1, 2, 3, 4}.

    • O(5):

      Since (56) is the only transposition that includes 5, and it is disjoint from all other transpositions in G, O(5) = {5, 6}.

    • G1:

      G1 = {ε, β}.

    • G2:

      G2 = {ε, αβα}.

    • G4:

      G4 = {ε, β}.

    • G5:

      G5 = {ε, β}.

  4. Let f,g ∈ G. Prove that f and g are in the same left coset of Gu iff f(u) = g(u).

    If Assume that f(u) = g(u). Let f = p∘f', where f' ∈ G. Then, f ∈ pGu.

    Since f' ∈ Gu, f'(u) = u. f(u) = (p∘f')(u) = p(f'(u)) = p(u).

    Let g' = p-1∘g. Then, g'(u) = (p-1∘g)(u) = p-1(g(u)) = p-1(f(u)) = u. Therefore, g'(u) = u, and g' ∈ Gu. Since g' = p-1∘g, p∘g' = g. Therefore, g ∈ pGu, the same coset as f.

    Only If Assume f,g ∈ qGu. Then, ∃ f',g' ∈ Gu such that q∘f' = f and q∘g' = g. Since f',g' ∈ Gu f'(u) = u = g'(u).

    f(u) = (q∘f')(u) = q(f'(u)) = q(u). g(u) = (q∘g')(u) = q(g'(u)) = q(u). Therefore, f(u) = g(u).

  5. Using Exercise 4, show that |O(u)| = (G : Gu).

    Let α ∈ G. The left coset of Gu that contains α is determined by α(u), as shown in Exercise 4. The possible values of α(u) are exactly the elements of O(u), by the definition of O(u).

    Therefore, Gu has as many left cosets as O(u) has elements.

  6. Conclude from Exercise 5 that the size of every orbit (with respect to G) is a factor of |G|. In particular, if f ∈ Sa, the length of each cycle of f is a factor of the order of f in SA.

    Since |O(u)| = (G : Gu) = |G| / |Gu|, |G| = |Gu| × |O(u)|.

    If SA is finite, the set of powers of f is a subgroup of SA. It is closed under multiplication by its definition. If ord(f) = n, then f-1 = fn-1 ∈ SA, so it is also closed under inversion.

    Each disjoint cycle that is a factor of f is the orbit of any of its elements. Therefore, as shown above, its order is a factor of the order of f.
