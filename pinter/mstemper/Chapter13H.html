Problem H — Survey of All Eight-Element Groups

Let G be any group of order 8. If G has an element of order 8, then G ≘ Z8. We will assume that G has no elements of order 8, hence all elements ≠e have order 2 or 4.

In exercises 2-7, assume ∃ a ∈ G such that ord(a) = 4. Let H = ⟨a⟩ = {e, a, a2, a3}, and let b ∉ H.

  1. If every x ∈ G (x ≠ e) has order 2, let a, b, c be three such elements. Prove that G = {e, a, b, c, ab, ac, bc, abc} and conclude that G ≘ Z2 × Z2 × Z2.

    Lemma Any group having only elements with order under 3 is Abelian. First, we note that in such a group, every element is its own inverse. Then, selecting x and y from such a group, xy = (xy)-1 = y-1x-1 = yx.

    Now, we look at possible products in G. Pre-multiplying each element of {e, a, b, c, ab, ac, bc, abc} gives {ae, aa, ab, ac, aab, aac, abc, aabc}. Since each element is its own inverse, this is {a, e, ab, ac, b, c, abc, bc}, which is the same set. Post-multiplying each element of {e, a, b, c, ab, ac, bc, abc} by a gives the same result, since this group is Abelian (by the lemma).

    Multiplying each element of {e, a, b, c, ab, ac, bc, abc} by b gives {be, ba, bb, bc, bab, bac, bbc, babc}. Applying commutativity and the fact that each element is its own inverse gives {b, ab, e, bc, a, abc, c, ac}, which is the same set.

    Multiplying each element of {e, a, b, c, ab, ac, bc, abc} by c gives {ce, ca, cb, cc, cab, cac, cbc, cabc}. Applying commutativity and the fact that each element is its own inverse gives {c, ac, bc, e, abc, a, b, ab}, which is the same set.

    Therefore, no products other than those listed are possible, and G = {e, a, b, c, ab, ac, bc, abc}. As shown in Chapter 9, Problem F, Exercise 4, this group is isomorphic to Z2 × Z2 × Z2.

  2. Assume that there is in Hb (see introduction) an element of order 2. (Let b be this element.) If ba = a2b, prove that b2a = a4b2, hence a = a4, which is impossible. Conclude that either ba = ab or ba = a3b.

    ba a2b assumption
    b2a ba2b pre-multiply both sides by b
    b2a (ba)(ab) associativity
    b2a (a2b)(ab) ba = a2b
    b2a a2(ba)b associativity
    b2a a2(a2b)b ba = a2b
    b2a (a2a2)bb associativity
    b2a a4b2 simplify
    ea a4e ord(b) = 2
    a a4 identity
    a-1a a-1a4 pre-multiply both sides by a-1
    e ea3 inverse
    e a3 identity

    The assumption that ba = a2b leads to the result that a3 = e. However, this is not true, because 3 does not divide |G| = 8. (For that matter, it was stated that ord(a) = 4.)

    Since ord(a) = 4, possible values for ba are ab, a2b, a3b, eb. We have just shown that ba ≠ a2b. It cannot be true that ba = b, because that would imply a = e. Since ord(a) = 4, this cannot be true, either.

    Thus the only possible values of ba are ab or a3b.

  3. Continuing from Exercise 2 with ord(b) = 2, prove that ba = ab implies G ≘ Z4 × Z2.

    The assumption that ba = ab implies that b commutes with all powers of a, as shown below:

    ba ab assumption
    baa aba post-multiply both sides by a
    baa aab ba = ab
    baaa aaba post-multiply both sides by a
    baaa aaab ba = ab

    This being the case, we have the following isomorphism:

    Z4 × Z2 G
    (0,0) e
    (1,0) a
    (2,0) a2
    (3,0) a3
    (0,1) b
    (1,1) ab
    (2,1) a2b
    (3,1) a3b

  4. Continuing from Exercise 2 with ord(b) = 2, prove that ba = a3b implies G ≘ D4.

    The reference for D4 will be as it was shown in Chapter 5, Problem F, Exercise 2. This is unfortunately written differently, with a being used to represent the order 2 element, and b representing the order 4 element. Additionally, that exercise generally has the order 2 element written before the order 4 element.

    However, once one adjusts for the differences in notation, the G and D4 have the same generating equations:

    D4 G
    a2 = e b2 = e
    b4 = e a4 = e
    ba = ab3 ba = a3b

    The following table expressing the isomorphsim must be read very carefully to prevent confusion.

    D4 G
    e e
    b a
    b2 a2
    b3 a3
    a b
    ab ab
    ab2 a2b
    ab3 a3b

  5. Contrary to Exercise 2, assume that there is no element of order 2 in Hb. Prove that b2 = a2.

    The only possible orders of elements are 1 (only for the identity), 2, and 4. By assumption, there are no elements of order 2 in Hb. Therefore, all of its elements are of order 4.

    Squaring an element of order 4 yields an element of order 2. Since there are no elements of order 2 in Hb, it must be true that b2 ∈ H. Since ord(e) = 1, ord(a) = 4, ord(a2) = 2, and ord(a3) = 1, we must have b2 = a2.

  6. Continuing from 5, prove that if ba = ab then (a3b)2 = e, contradicting the assumption that ord(a3b) = 4. Follow by proving that if ba = a2b then a = b4a = e, which is impossible. Conclude that ba = a3b.

    This shows that the assumption that ba = a2b leads to a = e. Since ord(a) = 4, this assumption must be false.

    Since we've eliminated the possibilities that ba = ab or ba = ba = a2b, it must be true that if Hb has no elements with order 2, then ba = a3b.

  7. The equations a4 = b4 = e, a2 = b2, and ba = a3b completely determine G. In this case, it is the quaternion group, Q. Write the Cayley table of Q.

    e a a2 a3 b ab a2b a3b
    e e a a2 a3 b ab a2b a3b
    a a a2 a3 e ab a2b a3b b
    a2 a2 a3 e a a2b a3b b ab
    a3 a3 e a a2 a3b b ab a2b
    b b a3b a2b ab a2 a e a3
    ab ab b a3b a2b a3 a2 a e
    a2b a2b ab b a3b e a3 a2 a
    a3b a3b a2b ab b a e a3 a2

Therefore, the only groups of order 8 are: Z8, Z2 × Z2 × Z2, Z4 × Z2, D4, and Q.
