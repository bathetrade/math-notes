Problem G — Isomorphic Groups on the Set R

  1. G is the set {x∈R : x≠-1}, with the operation x*y = x+y+xy. Show that f(x) = x-1 is an isomorphism from R* (the multiplicative group of non-zero reals) to G.

    If z∈R*, then z≠0 and f(z)≠-1, and f(z)∈G. Let a,b ∈ R*. If a≠b then f(a) = a-1 ≠ b-1 = f(b), so f is one-to-one. Let p,q ∈ R*. If f(p)=f(q), then p-1=q-1, and p=q. Therefore f is onto, and hence a bijection.

    Let c,d ∈ R*. Then f(c)*f(d) = (c-1)*(d-1) = (c-1)+(d-1)+(c-1)(d-1) = c+d-2 + cd-c-d+1 = cd-1 = f(cd). Therefore, f preserves products and is an isomorphism.

  2. I am going to assume that the operation on R is intended to be addition.

    G is the set of reals with the operation x*y = x+y+1. Find an isomorphism f:R → G and prove that it is an isomorphism.

    Let f(z) = z-1. It is obvious that f is a bijection.

    If p,q ∈ R, then f(p)*f(q) = p-1 + q-1 + 1 = p+q-1 = f(p+q), and f preserves products.

    Since f is a bijection and preserves products, it is an isomorphism.

  3. G is the set of non-zero reals with the operation x*y = xy/2. Find an isomorphism from R* to G.

    Let f(z) = 2z. It is obvious that f is a bijection.

    If p,q ∈ R, then f(p)*f(q) = 2p*2q/2 = 2pq = f(pq), so f preserves products.

    Since f is a bijection and preserves products, it is an isomorphism.

  4. Show that f(x,y) = (-1)yx is an isomorphsim from R+ × Z2 to R*.

    If (w,z) ∈ (R+ × Z2), then since z∈{0,1}, (-1)z∈{-1,1}. Combining this with w∈R+ gives f(w,z)∈R*.

    Let (p,q),(r,s) ∈ (R+ × Z2). If (p,q)≠(r,s) we must have at least one of p≠r or q≠s. If p≠r then (-1)qp ≠ (-1)sr no matter the values of q and s, since p and q are both positive. If q≠s, then one of {(-1)qp, (-1)sr} is positive and the other is negative. Therefore, f is onto.

    Let (a,b),(c,d) ∈ (R+ × Z2). If f(a,b) = f(c,d), then they must have the same signs, which requires that b=d. Also, they must have the same magnitudes, which requires that a=c. Therefore, if f(a,b) = f(c,d), then (a,b) = (c,d), and f is one-to-one.

    Therefore, f is a bijection.

    If (w,x),(y,z) ∈ (R+ × Z2), then f(w,x)f(y,z) = (-1)xw(-1)zy = (-1)x+zwy = f((wy),(x+z)). Therefore, f preserves products.

    Since f is a bijection and preserves products, it is an isomorphism.
