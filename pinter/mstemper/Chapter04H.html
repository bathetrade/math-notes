Problem H — Powers and Roots of Group Elements

Let a,b ∈G. Prove the following:

  1. (bab-1)n = banb-1 ∀ n∈ Z+

    Base For n=1, (bab-1)1 = ba1b-1.

    Hypothesis Assume that for some positive k, (bab-1)k = bakb-1

    Step (bab-1)k+1 = (bab-1)k(bab-1) = bakb-1bab-1 = bakeab-1 = bakab-1 = bak+1b-1.

  2. If ab=ba, then (ab)n=anbn ∀ n∈ Z+

    Base For n=1, (ab)1 = ab = a1b1.

    Hypothesis Assume that for some positive k, (ab)k=akbk

    Step (ab)k+1 = (ab)k(ab) = (akbk)(ab) = akbkab = akabkb = ak+1bk+1.

  3. If xax=e, then (xa)2n=an ∀ n∈ Z+

    Base For n=1, (xa)2n = (xa)2 = xaxa = ea = a = a1.

    Hypothesis Assume that for some positive k, (xa)2k=ak.

    Step (xa)2(k+1) = (xa)2k+2 = (xa)2k(xa)2 = akxaxa = akea = aka = ak+1.
