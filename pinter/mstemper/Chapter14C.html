Problem C — Elementary Properties of Homomorphisms

Let G, H, and K be groups. Prove the following:

  1. If f:G → H and g:H → K are homomorphisms, then g∘f:G → K is a homomorphisms.

    Let x,y ∈ G. Since f is a homomorphism, f(x)f(y) = f(xy). Let p=f(x) and q=f(y). Since g is a homomorphism, g(p)g(q) = g(pq), which means that g(f(x))g(f(y)) = g(f(x)f(y)) = g(f(xy)), which can also be written (g∘f)(x)(g∘f)(y) = (g∘f)(xy).

    Therefore, g∘f preserves products and is a homomorphism.

  2. If f:G → H is a homomorphism with kernel K, then f is injective iff K = {e}.

    If Assume that K = {e}. Let f(x) = f(y). Then, e = (f(x))-1f(x) = (f(x))-1f(y). (f(x))-1f(x) = e = f(x-1x) = f(x-1)f(x), so (f(x))-1 = f(x-1). Since K = {e}, f(z) = e iff z=e. Since homomorphisms preserve products, e = (f(x))-1f(y) = f(x-1)f(y) = f(x-1y) means that x-1y = e. Since any element of a group has only one inverse, x-1 = y-1, and x=y. Therefore, f is injective.

    Only If If f is a homomorphism, then e ∈ K. If f(e) ≠ e, then f(a)f(e) ≠ f(a) = f(ae), and products are not preserved. Therefore, if K≠{e}, then ∃ x ∈ K such that x≠e. Thus, f(e) = e = f(x), and f is not injective.

  3. If f:G → H is a homomorphism and K is any subgroup of G, then f(K) = {f(x) : x ∈ K} is a subgroup of H.

    Let p,q ∈ K. Since K is a group, it is closed under multiplication, so pq ∈ K and f(pq) ∈ f(K). Since f is a homomorphism, f(p)f(q) = f(pq), so f(K) is closed under multiplication.

    Let a ∈ K. Since K is a group, it is closed under inversion, so a-1 ∈ K. Since f is a homomorphism, f(e) = f(aa-1) = f(a)f(a-1), which means that f(a-1) = (f(a))-1, and f(K) is closed under inversion.

  4. If f:G → H is a homomorphism and J is any subgroup of H, then f-1(J) = {x ∈ G : f(x) ∈ J} is a subgroup of G.

    Let a,b ∈ f-1(J). Then, f(a),f(b) ∈ J. Since J is a group, it is closed under multiplication. Therefore, f(ab) = f(a)f(b) ∈ J. Therefore, ab ∈ f-1(J), which means that f-1(J) is closed under multiplication.

    Let c ∈ f-1(J). Then, f(c) ∈ J. Since J is a group, it is closed under inversion. Therefore (f(c))-1 ∈ J. As shown in the previous exercise, (f(c))-1 = f(c-1), which means that f(c-1) ∈ J and c-1 ∈ f-1(J). Therefore, f-1(J) is closed under inversion.

    Since f-1(J) is closed under both multiplication and inversion, it is a subgroup.

    Also prove that ker(f) ⊂ f-1(J).

    Since J is a group, e ∈ J. If x ∈ G and f(x) = e, then (by definition) x ∈ ker(f). Therefore, x ∈ f-1(e) ⊂ f-1(J).

  5. If f:G → H is a homomorphism with kernel K, and J is a subgroup of G, let f1 designate the restriction of f to J. Then ker(f1) = J ∩ K.

    Let x ∈ ker(f1). Since ker(f1) ⊂ J, x ∈ J. Since f1(x) = e, f(x) = e, and x ∈ K.

    Therefore, if x ∈ ker(f1), then x ∈ J ∩ K.

    Therefore, ker(f1) ⊂ J ∩ K.

    Let y ∈ J ∩ K. Then, y ∈ J and y ∈ K. Since x ∈ J, x ∈ the range of f1. Since x ∈ K, f1(x) = f(x) = e.

    Therefore, if x ∈ J ∩ K, then x ∈ ker(f1).

    Therefore, J ∩ K ⊂ ker(f1).

    Since each is a subset of the other, ker(f1) = J ∩ K.

  6. For any group G, the function f:G → G given by f(x) = e is a homomorphism.

    Let a,b ∈ G. f(a)f(b) = ee = e = f(ab). Therefore f preserves products and is a homomorphism.

  7. For any group G, {e} and G are homomorphic images of G.

    The previous exercise showed that f(x) = e is a homomorphism. That this function is onto {e} is obvious. Therefore {e} is a homomorphic image of G.

    The identity function on G is a homomorphism, and it is onto G. Therefore, G is a homomorphic image of itself.

  8. The function f:G → G given by f(x) = x2 is a homomorphism iff G is Abelian.

    If Let a,b ∈ G. If G is Abelian, then f(a)f(b) = a2b2 = aabb = abab = (ab)2 = f(ab), and f preserves products, making it a homomorphism.

    Only If Let a,b ∈ G. If f is a homomorphism, then it preserves products, giving aabb = a2b2 = f(a)f(b) = f(ab) = (ab)2 = abab. In other words, if f is a homomorphism, then a and b commute. Since a and b were any elements of G, it is Abelian.

  9. The functions f1:G × H → G given by f1(x,y) = x, and f2:G × H → H, given by f2(x,y) = y are homomorphisms.

    f1 Let (x1,y1),(x2,y2) ∈ G × H. Then f1(x1,y1)f1(x2,y2) = x1x2, and f1((x1,y1)(x2,y2)) = f1(x1x2,y1y2) = x1x2. Therefore f1 preserves products and is a homomorphism.

    f2 Let (x1,y1),(x2,y2) ∈ G × H. Then f2(x1,y1)f2(x2,y2) = y1y2, and f2((x1,y1)(x2,y2)) = f2(x1x2,y1y2) = y1y2. Therefore f2 preserves products and is a homomorphism.
