Problem B — Elementary Properties of Cyclic Groups

Prove each of the following:

  1. If G is a group of order n, G is cyclic iff G has an element of order n.

    If Let a∈G have order n. {a0, a1, ..., an-1} are elements of G, by Theorem 3 of the previous chapter. Since there are n elements in this set, and G has order n, G cannot have any other elements, and ⟨a⟩ = G, which makes G cyclic.

    Only If If G is cyclic, then it has a generator, b. In this case, G = {b0, b1, ..., bn-1} which is n elements, so ord(b) = n.

  2. Every cyclic group is Abelian.

    Let G = ⟨a⟩. Select p,q∈G. Since G is generated by a, these may be rewritten as p=am and q=an.

  3. If G = ⟨a⟩ and b∈G, ord(b) | ord(a).

    Since a generates G, b may be written as am. Letting ord(a) = n, Chapter 10, Problem G, Exercise 5 shows that ord(b) = ord(am) = lcm(m,n)/m.

    From number theory, we know that lcm(m,n) = mn/gcd(m,n). Therefore, lcm(m,n)/m = n/gcd(m,n), which divides n = ord(a).

  4. In any cyclic group of order n, there are elements of order k for every (positive) integer k which divides n.

    Let a be a generator of a group with order n. For any positive integer k that divides n, let p = n/k. Then, by Chapter 10, Problem G, Exercise 5, ord(ap) = lcm(p,n)/p.

    From number theory (as shown in the previous exercise) we have lcm(p,n)/p = n/gcd(p,n). Since p = n/k and (n/k) | n, gcd(p,n) = n/k, giving us ord(ap) = n/(n/k) = k.

  5. Let G be an Abelian group of order mn, where m and n are relatively prime. If G has an element of order m and an element of order n, then G is cyclic.

    Let a be an element of G with ord(a) = m, and b an element with ord(b) = n. As shown in Chapter 10, Problem E, Exercise 4, since a and b commute and m and n are relatively prime, ord(ab) = ord(a)ord(b).

    Since |G| = ord(ab), G = ⟨ab⟩, and G is cyclic.

  6. Let ⟨a⟩ be a cyclic group of order n. If n and m are relatively prime, then f(x) = xm is an automorphism of ⟨a⟩.

    Let x,y∈⟨a⟩ such that x≠y. Let x=ap and y=aq. Then f(x) = xm = (ap)m = apm and f(y) = ym = (aq)m = aqm. This gives f(x) = f(y)a(p-q)m.

    This means that f(x) = f(y) iff a(p-q)m = e. Since m and n are relatively prime, this can only be true if (p-q) is a multiple of n, (p-q) = kn.
