Problem D — A Cyclic Group of Permutations

Define fn(x) = x+n for all n in Z.

  1. Prove that for each n, fn is a permutation on R.

    For any a∈R, fn(a-n) = a, so fn is onto R.

    If fn(a) = fn(b), then a+n = b+n, so we must have a=b. This shows that fn is one-to-one.

  2. Prove that fn∘fm = fn+m and that fn-1 = f-n.

    [fn∘fm](x) fn(fm(x)) definition of composition
    fn(x+m) definition of fm
    (x+m)+n definition of fn
    x+(m+n) associativity of addition
    x+(n+m) commutativity of addition
    [fn+m](x) definition of fn+m

    To show an inverse, we must first determine the identity element for this potential group. Since fn∘f0 = fn+0 = fn, f0 is the identity.

    f0 additivie inverse

  3. Let G = {fn | n∈Z}. Show that G is a sub-group of SR.

    For G to be a sub-group of SR:

    • Each element of G must be an element of SR. This was proved in Exercise 1 of this problem.
    • G must be closed under the group operation (composition in this case). This was proved in Exercise 2 of this problem.
    • G must be closed under inversion. This was proved in Exercise 2 of this problem.

  4. Prove that G is cyclic.

    Since, for any n∈N, f1n = fn, f1 generates G.
