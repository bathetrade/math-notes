Problem G — Properties Preserved Under Homomorphism

If f:G→H is a homomorphism onto H, prove each of the following.

  1. If G is Abelian, then H is Abelian.

    Let x,y ∈ G. Since f is a homomorphism, it preserves products. Since G is Abelian, xy = yx. Therefore, f(x)f(y) = f(xy) = f(yx) = f(y)f(x). Thus, every element in the image of G commutes with every other element in the image of G. Since f is onto H, the image of G is H, and H is Abelian. ∎

  2. If G is cyclic, then H is cyclic.

    Let y ∈ H. Since f is onto, ∃ x ∈ G such that f(x) = y.

    Let a be a generator of G. Since a is a generator, ∃ n ∈ ℕ+ such that an = x. Since f is a homomorphism, it preserves products, which means that (f(a))n = f(x) = y.

    Since the choice of y was arbitrary, f(a) generates all elements of H, making H cyclic. ∎

  3. If every element of G has finite order, every element of H has finite order.

    Let y ∈ H. Since f is onto, ∃ x ∈ G such that f(x) = y.

    Let |x| = n. Then, xn = e. Since f is a homomorphism, it preserves products, which means that yn = (f(x))n = f(xn) = f(e) = e. Therefore, |y| ∣ n and |y| is finite.

    Since the choice of y was arbitrary, every element of H has finite order. ∎

  4. If every element in G is its own inverse, every element in H is its own inverse.

    Let y ∈ H. Since f is onto, ∃ x ∈ G such that f(x) = y.

    Since every element of G is its own inverse, xx = e. Since f is a homomorphism, it preserves products and the identity. Therefore, yy = f(x)f(x) f(xx) = f(e) = e, and y is its own inverse.

    Since the choice of y was arbitrary, every element of H is its own inverse. ∎

  5. If every element of G has a square root, every element of H has a square root.

    Let y ∈ H. Since f is onto, ∃ x ∈ G such that f(x) = y.

    Since x has a square root, let aa = a2 = x. Since f is a homomorphism, it preserves products. Therefore, (f(a))2 = f(a)f(a) = f(aa) = f(x) = y. Thus, f(a) is the square root of a.

    Since the choice of y was arbitrary, every element of H has a square root. ∎

  6. If G is finitely generated, G is finitely generated.

    Let y ∈ H. Since f is onto, ∃ x ∈ G such that f(x) = y.

    Since G is finitely generated, let the number of its generators be n ∈ ℕ+ and call them g1, …, gn.

    Let x = g1p1…gnpn, where the pi ∈ ℕ. Then, y = f(x) = f(g1p1…gnpn) = f(g1p1)…f(gnpn) = (f(g1)p1)…(f(gn)pn).

    Therefore, y is finitely generated by the f(gi)s. Since the choice of y was arbitrary, every element of H is finitely generated. ∎

    Note that f(g1), …, f(gn) might not be the smallest possible generating set for H. It is, however sufficient to do so.
