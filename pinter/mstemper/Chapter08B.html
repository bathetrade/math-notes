Problem B — Powers of Permutations

If π is a permutation, π1=π, πn+1=πn∘π, and πn∘π-n=e.

In Exercises 2 through 8, α is a cycle of length "s".

  1. Compute α-1, α2, α3, α4, α2 for each of the following:

    1. α = (123)

      • α-1 = (321)
      • α2 = (321)
      • α3 = ()
      • α4 = (123)
      • α5 = (321)

    2. α = (12345)

      • α-1 = (54321)
      • α2 = (13524)
      • α3 = (14253)
      • α4 = (54321)
      • α5 = ()

    3. α = (123456)

      • α-1 = (654321)
      • α2 = (135)(246)
      • α3 = (14)(25)(36)
      • α4 = (153)(264)
      • α5 = (654321)

  2. Describe the distinct powers of α. How many are there?

    Each application of α moves an element one "to the right" in α. Since α has length "s", s applications of α will move each element "all the way around" until it has moved back to its starting point — itself. Therefore, αs=e, αs+1=α1, and so on. This gives us s distinct powers of α.

  3. Find the inverse of α and show that α-1=αs-1.

    If α = (a1, a2, ... as), then α-1 = (as, ... a2, a1).

    Since αs-1α1 = αs, and αs = e, αs-1 = α-1.

  4. Prove that α2 is a cycle if and only if s is odd.

    This may be generalized to "αn is a cycle if and only if gcd(s,n)=1."

    If α2 maps as-1 to a1, as to a2, and all other ai to ai+2. Since s is odd, this gives: (a1, a3, ... as, a2, ... as-1), which contains all elements of α.

    Only if If s is even, then α2 maps ai with i even to aj with j even, while ai with i odd are mapped to aj with j odd. This means that α2 is the product of two disjoint cycles, when s>2. (When s=2, then α2 = α, which is a cycle.)

  5. Prove that if s is odd, α is the square of a cycle of length s.

    Let p = (s+1)/2. Then, the square of (a1, ap+1, a2, ap+2, ... as, ap) is α.

  6. Prove that if s is even, α2 is the product of two cycles of length s/2.

    The two cycles are (a1, a3, ... as-1) and (a2, a4, ... as).

  7. Prove that if s=kt, then αk is the product of k cycles of length t.

    αk maps ajk to a(j+1)k when j<t and as to ak, which is a cycle of length t. Simlarly, the elements ajk-q form a cycle for any q<k. This gives k cycles, each of which is of length t.

  8. Prove that if s is a prime number, every power of α is a cycle.

    Label the elements of α as "a0", "a1", ... "as-1", so that α(ai) = ai+1 when i<(s-1) and α(as-1) = a0.

    Pinter does not (as far as I can tell) refer to the identity function as a "one-cycle". In fact, his statement of Theorem 1 says: "[...] either the identity, a single cycle, or a product of disjoint cycles." The wording seems to treat these as mutually-exclusive cases.

    However, since αs is the identity (whether or not s is prime), we must be expected to consider the identity a one-cycle.

    Define the function m(i) as the remainder when i is divided by s. (In programming languages, this would be "mod(i,s)" or "i%s".)

    With these in place, αk(ai) = am(ki). One factor of α is then (am(k)am(2k)...a0).

    Since s is prime, lcm(s,k)=sk if k is not a multiple of s. This means that the smallest positive i for which m(ik)=0 is s. Therefore this cycle has s elements, and αk has no other factors and is a cycle.

    On the other hand, if k is a multiple of s, say k=st, then for any p (0≤p≤s), αk(ap) = αst(ap) = αs∘αs...αs(ap). ("t" applications of αs)

    Since αs(p) = p, this means that αst is the identity function, which is a one-cycle.
