Problem J — Regular Representation of Groups

Find the right and left regular representation (or the regular representation for Abelian groups) and compute their tables (or its table):

  1. The group P2 of subsets of a two-element set (as described in Chapter 3, Problem C).

    The operation (symmetric difference) was previously shown to be commutative, so this is an Abelian group, and only has one regular representation.

    π0 = 
    ∅ {a} {b} {a,b}
    ∅ {a} {b} {a,b}
    π1 = 
    ∅ {a} {b} {a,b}
    {b} {a,b} ∅ {a}
    π2 = 
    ∅ {a} {b} {a,b}
    {a} ∅ {a,b} {b}
    π3 = 
    ∅ {a} {b} {a,b}
    {a,b} {b} {a} ∅
    ∘ π0 π1 π2 π3
    π0 π0 π1 π2 π3
    π1 π1 π0 π3 π2
    π2 π2 π3 π0 π1
    π3 π3 π2 π1 π0

  2. Z4

    Since addition is commutative, this is an Abelian group and only has one regular representation.

    π0 = 
    0 1 2 3
    0 1 2 3
    π1 = 
    0 1 2 3
    1 2 3 0
    π2 = 
    0 1 2 3
    2 3 0 1
    π3 = 
    0 1 2 3
    3 0 1 2
    + π0 π1 π2 π3
    π0 π0 π1 π2 π3
    π1 π1 π2 π3 π4
    π2 π2 π3 π0 π1
    π3 π3 π0 π1 π2

  3. The group G of matrices described on pages 28 and 29 of the text.

    As shown in the text, this group is non-Abelian, so both a left regular representation (G*) and a right regular representation (Gρ) will be needed.

    The left regular representation (G*) is:

    πI = 
    I A B C D K
    I A B C D K
    πA = 
    I A B C D K
    A I C B K D
    πB = 
    I A B C D K
    B K D A I C
    πC = 
    I A B C D K
    C D K I A B
    πD = 
    I A B C D K
    D C I K B A
    πK = 
    I A B C D K
    K B A D C I

    The Cayley table for the left regular representation is:

      πI πA πB πC πD πK
    πI πI πA πB πC πD πK
    πA πA πI πC πB πK πD
    πB πB πK πD πA πI πC
    πC πC πD πK πI πA πB
    πD πD πC πI πK πB πA
    πK πK πB πA πD πC πI

    The right regular representation (Gρ) is:

    πI = 
    I A B C D K
    I A B C D K
    πA = 
    I A B C D K
    A I K D C B
    πB = 
    I A B C D K
    B C D K I A
    πC = 
    I A B C D K
    C B A I K D
    πD = 
    I A B C D K
    D K I A B C
    πK = 
    I A B C D K
    K D C B A I

    The Cayley table for the right regular representation is:

      πI πA πB πC πD πK
    πI πI πA πB πC πD πK
    πA πA πI πK πD πC πB
    πB πB πC πD πK πI πA
    πC πC πB πA πI πK πD
    πD πD πK πI πA πB πC
    πK πK πD πC πB πA πI

    Unsurprisingly, the two Cayley tables are transposes of each other.
