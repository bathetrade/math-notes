Problem D — Further Properties of Order

Let a be any element of finite order of a group G. Prove the following:

  1. If ap = e where p is a prime number, ord(a) = p. (a≠e).

    By Theorem 5, ap = e implies that p is a multiple of ord(a). Since p is prime, it is only a multiple of itself and 1. The only element having order 1 is e, which is excluded by the problem statement. Therefore, ord(a) = p.

  2. The order of ak is a divisor of the order of a.

    Looking again at kp = lcm(k,m), we see that p = lcm(k,m)/k. This proves my hunch from Problem B.

    Let ord(a) = m and ord(ak) = p. e = (ak)p = akp. By Theorem 5, kp is a multiple of ord(a) = m. In fact, kp must be lcm(k,m), because raising a to any multiple of m will yield e.

    kp = lcm(k,m) = km/gcd(k,m). p = m/gcd(k,m). gcd(k,m)p = m. Therefore, p|m (or: the order of ak divides the order of a).

  3. If ord(a) = km, ord(ak) = m.

    Ord(a) = km implies that e = akm = (ak)m. If 0<x<m, (ak)x = akx ≠ e, because kx<km, and ord(a) = km implies that km is the smallest positive integer such that akm = e.

    Therefore, ord(ak) = m.

  4. If ord(a) = n and n is odd, then ord(a2) = n.

    As shown in Exercise 2, ord(ak) = lcm(k,ord(a))/k. In this case, k=2 and ord(a)=n. lcm(2,n) = 2n when n is odd, so lcm(2,n)/2 = n.

  5. If ord(a) = n and ar = as, then n|(r-s).

    as ar Given
    ar-s+s additive inverse
    a(r-s)+s associativity
    a(r-s)as xp+q = xpxq

    Since as = a(r-s)as, it must be true that a(r-s) = e. By Theorem 5, n|(r-s).

  6. If a is the only element of order k in G, then a is in the center of G.

    As shown in Problem C, Exercise 4, ord(a) = ord(bab-1) for all b. If a is the only element of G that has order k, then it must be the case that a = bab-1 for all b.

    a bab-1 a is only element with order k
    ab bab-1b post-multiply by b
    ab bae inverses
    ab ba identity

    Therefore, a commutes with every b, and is by definition an element of the center of G.

  7. If ord(a) is not a multiple of m, then ord(ak) is not a multiple of m.

    In Exercise 2, it was shown that ord(ak) | ord(a). Another was to express this is that there exists a natural number p such that p×ord(ak) = ord(a).

    If ord(ak) is a multiple of m, then there is a natural number q such that qm = ord(ak). This means that p×ord(ak) = pqm = ord(a), or ord(a) is a multiple of m.

    Therefore, if ord(a) is not a multiple of m, ord(ak) cannot be a multiple of m either.

  8. If ord(a) = mk and ark = e, then r is a multiple of m.

    Remember that, since ord(x) is the least positive integer such that xord(x) = e, ord(e) = 1, not 0.

    By Theorem 5, rk must be a multiple of mk. In other words, there is some positive integer q such that rk = qmk. Dividing both sides by k (which is positive), we get r = qm, or r is a multiple of m.
