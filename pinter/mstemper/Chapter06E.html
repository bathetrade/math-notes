Problem E — Inverses of Functions

Each of the following functions is bijective. Describe its inverse.

  1. f:(0,∞)→(0,∞), defined by f(x)=1/x.

    f-1(x) = f(x).

  2. f:R→(0,∞), defined by f(x)=ex.

    f-1(x) = ln(x).

  3. f:R→R, defined by f(x)=x3+1.

    f-1(x) = (x-1)1/3.

  4. f:R→R, defined by:

    f(x) 2x if x is rational
    3x if x is irrational
    f-1(x) x/2 if x is rational
    x/3 if x is irrational

  5. A={a,b,c,d}, B={1,2,3,4}, and f:A→B is given by:

    abcd
    3124

    f-1:B→A is given by:

    1234
    b2ad

  6. G is a group, a∈G, and f:G→G is given by f(x)=ax.

    f-1(x) = a-1x.
