Problem E — Elementary Properties of Cosets

Let G be a group, and H a subgroup of G. Let a and b denote elements of G. Prove the following:

  1. Ha = Hb iff ab-1 ∈ H.

    If Assume that ab-1 ∈ H. Then a ∈ Hb. Since cosets partition a group, this means that Ha = Hb.

    Only If Assume that Ha = Hb. Let x ∈ H. Then, xa ∈ Hb and xab-1 ∈ H. Since x ∈ H, this means that ab-1 ∈ H.

  2. Ha = H iff a ∈ H.

    Let b=e, in which case H = He = Hb, and b-1=e. Then the result just proved is applicable.

  3. If aH = Ha and bH = Hb, then (ab)H = H(ab).

    Note that neither a nor b is necessarily in H so we cannot say that (ab)H = H(ba).

    Since a commutes with every element of H, and b commutes with every element of H, (ab) commutes with every element of H. For instance, if x ∈ H, abx = axb = xab.

  4. If aH = Ha, then a-1H = Ha-1.

    Since a commutes with every element of H, a-1 does. Let x ∈ H:

    ax xa a commutes with every element of H
    a-1ax a-1xa pre-multiply both sides by a-1
    a-1axa-1 a-1xaa-1 post-multiply both sides by a-1
    exa-1 a-1xe inverses
    xa-1 a-1x identity

  5. If (ab)H = (ac)H, then bH = cH.

    Let x ∈ H:

    (ab)x (ac)x from hypothesis
    a(bx) a(cx) associativity
    a-1a(bx) a-1a(cx) pre-multiply by a-1
    e(bx) e(cx) inverse
    (bx) (cx) inverse

    Since (ab)x = (ac)x ⇒ (bx) = (cx) ∀ x ∈ H, (ab)H = (ac)H ⇒ bH = cH.

  6. The number of right cosets of H is equal to the number of left cosets of H.

    G must be a finite group. Otherwise, |G| / |H| is not defined.

    First, we'll prove that the left cosets of H are a partition of G. Then, we'll prove that there is a bijection between H and any left coset of H.

    The combination of these facts implies that the number of cosets of H is equal to |G| / |H|, which is the same as the number of right cosets.

    Theorem i The family of all left cosets aH, as a ranges over G, is a partition of G.

    First, we must show that any two left cosets, say bH and cH, are either disjoint or equal.

    If they're disjoint, we're done.

    If not, let x ∈ bH ∩ cH. Since x ∈ bH, x = bh1 for some b1 in H. Similarly, x = ch2 for some h2 in H. Thus, bh1 = ch2. Solving for b gives b = ch2h1-1. Therefore, c ∈ bH.

    Next, we must show that every element d ∈ G is in some left coset of H. Since d = de and e ∈ H, d ∈ dH.

    Since the left cosets are disjoint sets and every element of G is in one of them, they form a partition of H.

    Theorem 2i If aH is any left coset of H, there is a bijection between H and aH.

    Let f:H → aH be defined as f(x) = ax.

    For any y,z ∈ H, let f(y) = f(z). Then ay = az, a-1ay = a-1az, ey = ez, and y =z. Therefore f is one-to-one.

    Let p ∈ aH. Then, p = aq for some q ∈ H. Thus, p = f(q), and f is onto.

    Since f is one-to-one and onto, it is a bijection. Since there is a bijection between H and any left coset of H, H and its left cosets are all of the same cardinality.

    If G is a finite group, then the number of left cosets of H is |G| / |H|. This is the same as the number of right cosets of a finite group.

  7. If J and K are subgroups of G such that J = H ∩ K, then for any a ∈ G, Ja = Ha ∩ Ka.

    Let x ∈ Ja. Then, ∃ y ∈ J such that x = ya. If y ∈ J, then y ∈ H and y ∈ K. Then, ya ∈ Ha and ya ∈ Ka. Hence, x = ya ∈ Ha ∩ Ka, and Ja ⊂ Ha ∩ Ka.

    Let x ∈ Ha ∩ Ka. Then, x ∈ Ha, which means that for some y ∈ H, x = ya. It is also true that x ∈ Ka, which means that for some z ∈ K, x = za. However, ya = x = za iff y=z. So y ∈ H and y ∈ K, implying that y ∈ H ∩ K = J. Since y ∈ J, ya ∈ Ja, which means that Ha ∩ Ka ⊂ Ja.

    Since each is a subset of the other, Ja = Ha ∩ Ka.

    If (G : H) and (G : K) are finite, then (by Problem D, Part 1) (G : H)·(H : J) = (G : J) = (G : K)·(K : J). Therefore, (G : J) is finite.
