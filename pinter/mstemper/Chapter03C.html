Problem C — Groups of Subsets of a Set

Let A + B be the symmetric difference of sets A and B. The author shows that this operation is commutative and associative. Using P(D) (for an arbitrary set D) as the domain, examine the operation as described.

  1. Does this operation have an identity element?

    A + ∅ = (A - ∅) ∪ (∅ - A) = A ∪ ∅ = A.
    ∅ + B = (∅ - B) ∪ (B - ∅) = ∅ ∪ B = B.
    The empty set is the identity element for this operation.

  2. Does each element of P(D) have an inverse with respect to this operation?

    Let Q be an element of P(D). Then Q + Q = (Q - Q) ∪ (Q - Q) = ∅ ∪ ∅ = ∅.
    Each set is its own inverse.
