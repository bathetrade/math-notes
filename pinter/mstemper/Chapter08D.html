Problem D — Disjoint Cycles

In each of the following, let α and β be disjoint cycles, with α = (a1a2...as) and β = (b1b2...br).

Prove exercises 1-3:

  1. For every positive integer n, (αβ)n = αnβn.

    Note that since α and β are disjoint, they commute.

    Base (αβ)1 = αβ = α1β1.

    Hypothesis Assume that (αβ)k = αkβk.

    Step (αβ)k+1 = (αβ)k(αβ) = αkβkαβ. Since α commutes with β, it commutes with k βs, which means that αkβkαβ = αkαβkβ = αk+1βk+1.

  2. If αβ=e then α=e=β.

    Since α and β are disjoint cycles, α does not move any of the "b"s. This means that if β moves any "b"s, αβ can not be e. Therefore β must not move any "b"s and must be the identity.

    Since α and β are disjoint cycles, β does not move any of the "a"s. This means that if α moves any "a"s, αβ can not be e. Therefore α must not move any "a"s and must be the identity.

  3. If (αβ)t=e, then αt=e and βt=e.

    As shown in Exercise 1, (αβ)t = αtβt.

    Since α and β are disjoint cycles, αt and βt are disjoint cycles. As shown in Exercise 2, αtβt = e means that αt = e = βt.

  4. Find a transposition γ such that αβγ is a cycle.

    Let γ = (a1b1). Then, αβγ = (a1b2 ... brb1a2 ... as), which is a cycle.

    Note that (a1a2 ... as) = (ai+1 ... asa1 ... ai). Therefore, any element of α could be used instead of a1. Also, since (b1b2 ... br) = (bj+1 ... brb1 ... bj), any element of β could be used instead of b1.

  5. Let γ be the same transposition as in the preceeding exercise. Show that αγβ and γαβ are cycles.

    αγβ = (a1b1 ... bra2 ... as), which is a cycle. γαβ = (a1... asb1 ... br), which is a cycle.

  6. Why did he put this exercise (about non-disjoint cycles) in a problem titled "Disjoint Cycles"?

    Let α and β be cycles of odd length (not disjoint). Prove that if α2=β2 then α=β.

    Since |α| is odd, α2 is a cycle, as shown in Problem B, Exercise 4. Let α2 = (a0a1...an).

    For α2(ai) to be equal to aj, we must have α(α(ai)) = aj. In a cycle, α(ai)=α(ai+1) when i<n and α(ai)=α(a0) when i=n. This means that there is exactly one choice for α, and therefore β=α.
