Problem F — Functions on Finite Sets

  1. If the members of a committee vote for an officer, and each vote differs from all others, describe the range of the function f(x) that maps each member, x, to their vote.

    Since each member votes for a different person, the number of unique votes is the same as the number of members. Since the votes are in the same set as the voters, and the number of votes is the same as the number of voters, the range of votes must be the entire set of voters.

  2. Let A be a finite set. Explain why any injection f:A→A must be a surjection.

    If f is an injection, then |f(A)| = |A|. Since A is finite, it has no proper subsets with the same size as itself (assuming the Axiom of Choice). Therefore, the set of elements of A that are mapped to is exactly A itself, and f is a surjection.

  3. Let A be a finite set. Explain why any surjection f:A→A must be an injection.

    Assume that f is not an injection. Then, ∃a,b∈A such that f(a)=f(b). In this case, since A is finite, there must be some c∈A that is not in the image of f, which makes f not a surjection. Therefore, if f is a sujection, it must be an injection.

  4. Are the statements in exercises 2 and 3 necessarily true when A is infinite?

    Let A=N, and let f(x)=2x. This is an injection, since a≠b implies 2a≠2b. However, it is not a surjection, since ∀c∈N, 2c≠3.

    Let A=N, and let f(x)=

    f(x) x/2 if x is even
    (x-1)/2 if x is odd

    This is a surjection, since for any p∈N, f(2p)=p. However, it is not an injection, since for any any p∈N, f(2p+1)=f(2p).

  5. If |A|=n, how many functions f:A→A are there? How many of these are bijections?

    There are nn functions, and n! of these are bijections.
