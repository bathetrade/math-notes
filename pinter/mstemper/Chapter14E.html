Problem E — Further Properties of Normal Subgroups

Let G denote a group, and H a subgroup of G. Prove the following:

  1. If (G : H) = 2, then H is normal.

    Let a ∈ G. Then either a ∈ H or a ∉ H.

    If a ∈ H, then aH = H = Ha.

    If a ∉ H, then aH ≠ H ≠ Ha. Since H only has one coset in G, it must be true that aH = Ha.

    Since in all cases, we have aH = Ha, by Problem D, Exercise 5, H is normal.

  2. Suppose a ∈ G and ord(a) = 2. Then ⟨a⟩ is a normal subgroup of G iff a is in the center of G.

    If If a is in the center of G, then xa = ax for all x ∈ G. Then, the conjugate of a with respect to x is xax-1 = axx-1 = ae = a ∈ ⟨a⟩. Therefore, ⟨a⟩ is normal.

    Only If Since ord(a) = 2, ⟨a⟩ = {e,a}. If ⟨a⟩ is normal, then xax-1 ∈ {e,a}. If xax-1 = e, then xa = x, or a = e, which contradicts the condition that ord(a) = 2. Therefore, if ⟨a⟩ is normal, xax-1 = a ∀ x ∈ G. If xax-1 = a, then xa = ax. Since xe = ex, all elements of ⟨a⟩ commute with all elements of G. Therefore ⟨a⟩ is in the center of G.

  3. If a is any element of G, ⟨a⟩ is a normal subgroup of G iff a has the following property: ∀ x ∈ G, ∃ k ∈ Z+ such that xa = akx.

    If If xa = akx, then a = x-1akx = (x-1)ak(x-1)-1. Therefore, every conjugate of every power of a (every element of ⟨a⟩) is a, and ⟨a⟩ is normal.

    Only If If ⟨a⟩ is normal, then xax-1 = b ∈ ⟨a⟩. If b ∈ ⟨a⟩, then b = ak for some k ∈ Z+. Therefore, xax-1 = ak, which is equivalent to xa = akx.

  4. In a group G, a commutator is any product of the form aba-1b-1 (a,b ∈ G). If a subgroup H of G contains all of the commutators of G, then H is normal.

    I will prove the contrapositive. That is, if H is not normal, then H does not contain all of the commutators of G.

    First, a lemma. If H is a subgroup of G, with b ∈ H and a ∈ (G-H), then ba ∉ H.

    If ba ∈ H, then b-1ba ∈ H. But, b-1ba = ea = a ∉ H.

    If H is not normal, then for some h ∈ H and some x ∈ G, x-1hx ∉ H. If x-1hx ∉ H, then h-1x-1hx ∉ H.

    Since h-1x-1hx is a commutator of G, the contrapositive has been shown. Since the contrapositive of a statement is logically equivalent to the statement itself, the original statement has been proved.

  5. If H and K are subgroups of G, and K is normal, then HK is a subgroup of G.

    First, a lemma. If H < G and K ⊲ G, then KH ⊆ HK.

    Let h ∈ H and k ∈ K. Since H ⊆ G, h ∈ G. Since K is normal, h-1kh ∈ K.

    Since h-1kh ∈ K, h(h-1kh) = (hh-1)kh = ekh = kh ∈ HK. Since the choices of h and k were arbitrary, KH ⊆ HK.

    To show that a subset of a group is a subgroup, all that is necessary is to show that it is closed under inversion and multiplication.

    Inversion Let h ∈ H and k ∈ K. Then hk ∈ HK.

    Since K is a subgroup, it is closed under inversion and k-1 ∈ K. Since H is a subgroup, it is closed under inversion and h-1 ∈ H. Therefore, (hk)-1 = k-1h-1 ∈ KH ⊆ HK (by the lemma), and HK is closed under inversion.

    Multiplication Let h1, h2 ∈ H and k1, k2 ∈ K. Then h1k1, h2k2 ∈ HK.

    Multiplying these elements of HK gives h1k1h2k2 = h1(k1h2)k2. By the lemma, (k1h2) ∈ KH ⊆ HK.

    Thus, ∃ h3 ∈ H and k3 ∈ K such that h3k3 = k1h2. This gives h1(k1h2)k2 = h1(h3k3)k2 = (h1h3)(k3k2) ∈ HK, and HK is closed under multiplication.

    Since HK is closed under inversion and multiplication, it is a subgroup of G. ∎

  6. Let S be the union of all cosets Ha such that aH = Ha. Then S < G and H ⊲ S.

    To show that S < G, it is necessary (and sufficient) to show that it is closed under multiplication and inversion.

    Multiplication Let x, y ∈ G such that xH = Hx and yH = Hy. Then, x, y ∈ S, since x = ex ∈ Hx and y = ey ∈ Hy.

    Since yH = Hy and xH = Hx, (xy)(H) = x(yH) = x(Hy) = (xH)y = (Hx)y = H(xy).

    Therefore, if x,y ∈ S then xy ∈ S and S is closed under multiplication.

    Inversion Let x ∈ G such that xH = Hx. Then, x-1H = x-1He = x-1H(xx-1) = x-1(Hx)x-1 = x-1(xH)x-1 = (x-1x)Hx-1 = eHx-1 = Hx-1.

    Therefore, if x ∈ S then x-1 ∈ S, and S is closed under inversion.

    Since S is a subset of G that is closed under multiplication and inversion, S < G.

    Since H and S are groups, and H ⊆ S, H < S. By Problem D, Exercise 5, H ⊲ S. ∎
