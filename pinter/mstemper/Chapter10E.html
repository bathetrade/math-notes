Problem E — Relationship Between ord(ab), ord(a), and ord(b)

Let a and b be elements of a group G. Let ord(a) = m and ord(b) = n. Prove parts 1-5:

  1. If a and b commute, then ord(ab) | lcm(m,n).

    If a = b-1, then ab=e and ord(ab) = 1, which trivially divides lcm(m,n).

    Otherwise, let q = ord(ab). Then (ab)q = e. Since a and b commute, e = (ab)q = aqbq. Since a≠b-1, aq ≠ (bq)-1.

    Therefore, we must have aq=e and bq=e. This is true for a if q is a multiple of m, and it is true for b if q is a multiple of n. Therefore, it is true for both a and b if q is any multiple of m and n. This includes lcm(m,n), which trivially divides lcm(m,n).

    Therefore, whether or not a=b-1, ord(ab) | lcm(m,n).

  2. If m and n are relatively prime, no power of a can be equal to any power of b (except for apm = e = bqn).

    Select any integers r and s such that ar = bs = x. By Exercise 2 of Problem D, ord(x) = ord(ar) | ord(a), and ord(x) = ord(bs) | ord(b).

    Since ord(x) divides both ord(a) and ord(b), but ord(a) and ord(b) are relatively prime, it must be true that ord(x) = 1, which means that x = e.

  3. If m and n are relatively prime, then the products aibj (0≤i<m, 0≤j<n) are all distinct.

    By Theorem 3, ai (0≤i<m) are all distinct, as are bj (0≤j<n). By the previous exercise, all of the ai are distinct from all of the bj (except for a0 = e = b0), which means that all of the products must be distinct, as well.

  4. Let a and b commute. If m and n are relatively prime, ord(ab) = mn.

    Since ab=ba, (ab)k = akbk. This means that (ab)k = e whenever k is a multiple of both m and n, because amxbny = (am)x(bn)y = exey = e. Therefore, the maximum value that ord(ab) can take is lcm(m,n). Since m and n are relatively prime, lcm(m,n) = mn.

    By the Division Algorithm, k may be written as pm+q (0≤q<m) or rn+s (0≤s<n). Since (ab)pm = e = (ab)rn, this means that (ab)k = aqbs (since ab = ba). As shown in the previous exercise, these products are all distinct, giving mn possible products.

    Therefore, mn is not only the maximum possible value of ord(ab), but the minimum possible value, and ord(ab) = mn.

  5. Let a and b commute. Then there is an element c in G such that ord(c) = lcm(m,n).

    In the previous exercise, it was shown that ord(ab) = lcm(m,n) when ab = ba.

  6. Show that if a and b do not commute, it is not necessarily true that ord(ab) | lcm(m,n).

    The group G of matrices described on pages 28 and 29 of the text will be used.

    C2 = I = K2, so ord(C) = ord(K) = 2. The lcm(ord(C),ord(K)) = 2.

    If we look at the powers of CK = B, they are:

    B1 B
    B2 D
    B3 I

    So, ord(CK) = ord(B) = 3 ≠ 2 = lcm(2,2) = lcm(ord(C),ord(K)).
