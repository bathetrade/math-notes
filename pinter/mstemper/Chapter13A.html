Problem A — Examples of Cosets in Finite Groups

In each of the following, H < G. In exercises 1-5, list the cosets of H.

  1. G = S3, H = {ε, β, δ}.

    • Hε = Hβ = Hδ = {ε, β, δ} = H.
    • Hα = Hκ = Hγ = {α, κ, γ}.

  2. G = S3, H = {ε, α}.

    • Hε = Hα = {ε, α} = H.
    • Hβ = Hγ = {β, γ}.
    • Hδ = Hκ = {δ, κ}.

  3. G = Z15, H = ⟨5⟩.

    • H+0 = H+5 = H+10 = {0, 5, 10} = H.
    • H+1 = H+6 = H+11 = {1, 6, 11}.
    • H+2 = H+7 = H+12 = {2, 7, 12}.
    • H+3 = H+8 = H+13 = {3, 8, 13}.
    • H+4 = H+9 = H+14 = {4, 9, 14}.

  4. G = D4, H = {R0, R4}.

    • H∘R0 = H∘R4 = {R0, R4} = H.
    • H∘R1 = H∘R7 = {R1, R7}.
    • H∘R2 = H∘R5 = {R2, R5}.
    • H∘R3 = H∘R6 = {R3, R6}.

  5. G = S4, H = A4.

    • H() = H(132) = H(142) = H(123) = H(124) = H(12)(34) = H(143) = H(13)(24) = H(134) = H(14)(23) = H(243) = H(234) = {(), (132), (142), (123), (124), (12)(34), (143), (13)(24), (134), (14)(23), (243), (234)} = H.
    • H(12) = H(23) = H(24) = H(13) = H(14) = H(34) = H(1243) = H(1423) = H(1234) = H(1324) = H(1432) = H(1342) = {(12), (23), (24), (13), (14), (34), (1243), (1423), (1234), (1324), (1432), (1342) }.

  6. Indicate the order and index of each of the above subgroups.
