Problem A — Examples of Injective and Surjective Functions

For each of the following functions f:R→R, determine whether it is injective or surjective.

  1. f(x) = 3x + 4

    This function is injective. Assume that f(a)=f(b) for some a, b. Then:

    f(a) f(b) given
    3a+4 3b+4 definition of f
    3a 3b add -4 to both sides
    a b multiply both sides by 1/3

    This function is surjective. Let y∈R. Then, ((y-4)/3)∈R.

    f((y-4)/3) 3((y-4)/3)+4 definition of f
    y-4+4 multiplicative inverse
    y additive inverse

  2. f(x) = x3 + 1

    This function is injective. Assume that f(a)=f(b) for some a, b. Then:

    f(a) f(b) given
    a3+1 b3+1 definition of f
    a3 b3 add -1 to both sides
    a b raise both sides to the 1/3 power

    This function is surjective. Let y∈R. Then, (y-1)1/3∈R.

    f((y-1)1/3) ((y-1)1/3)3+1 definition of f
    y-1+1 multiplicative inverse
    y additive inverse

  3. f(x) = |x|

    Since f(1) = |1| = 1 = |-1| = f(-1), this function is not injective.

    Since there is no x∈R such that |x|=-1, this function is not surjective.

  4. f(x) = x3 - 3x

    This function is not injective. f(-sqrt(3)) = f(0) = f(sqrt(3)).

    df/dx = 3x2-3. Since this is strictly positive for x>sqrt(3), every positive real is the image of some point in (sqrt(3),∞). Since this is strictly negative for x<-sqrt(3), every negative real is the image of some point in (-∞,-sqrt(3)). We have already shown that zero is the image of three points, so every real is the image of some point in R.

  5. f(x) x if x is rational
    2x if x is irrational

    First note that the product of any irrational and any non-zero rational is irrational. This implies that f(x) is irrational for any irrational x.

    This function is injective. If f(a)=f(b) is rational, then a=b. If f(a)=f(b) is irrational, then 2a=2b, and a=b.

    This function is surjective. If y is any rational, f(y)=y. If y is any irrational, f(y/2)=y.

  6. f(x) 2x if x is an integer
    x if x is non-integral

    This function is injective. If f(a)=f(b) is an even integer, then 2a=2b and a=b. If f(a)=f(b) is not an integer, then a=b.

    This function is not surjective, because there is no x that maps to any odd integer.

  7. Determine the range of each of the functions in the previous exercises.

    1. Since f(x)=3x+4 is surjective, its range is R.

    2. Since f(x)=x3+1 is surjective, its range is R.

    3. The range of the absolute value function is R+ ∪ {0}.

    4. Since f(x)=x3-3x is surjective, its range is R.

    5. Since {f(x)=x (x is rational)} ∪ {f(x)=2x (x is irrational)} is surjective, its range is R.

    6. The range of {f(x)=2x (x is integer)} ∪ {f(x)=x (x is non-integral)} is R - {2x+1 | x∈Z}.
