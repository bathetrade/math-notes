Problem F — Order of Cycles

  1. Prove in Sn: If α = (a1 ... as) is a cycle of length s, then αs = e, α2s = e, and α3s = e. Is αk = e for any positive integer k<s? (Explain.)

    This would be a lot simpler if he'd number elements starting with zero instead of one!

    By the definition of α, α(aj) = aj+1 when j+1≤s and α(aj) = aj+1-s when j+1>s. This may be generalized to αk(aj) = aj+k when j+k≤s and αk(aj) = aj+k-s when s<j+k≤2s.

    Specifically, in the case where k=s, we have αk(aj) = aj+k-s = aj. Since αs(aj) = aj for all j, αs = e.

    α2s = αs∘αs = e∘e = e.

    If α is any permutation, the least positive integer n such that αn = e is called the order of α.

    α3s = αs∘α2s = e∘e = e.

    If 0<k<s, then αk(aj) = aj+k when j+k≤s and αk(aj) = aj+k-s when j+k>s (as above). In the first case, j+k>j, which means that αk(aj) ≠ aj. In the second case, since k<s, j+k-s<j and again αk(aj) ≠ aj.

  2. Prove in Sn: If α = (a1 ... as) is any cycle of length s, then the order of α is s.

    This follows immediately from the first and last parts of Exercise 1.

  3. Find the order of each of the following permutations:

    1. (12)(345)

      The order is 6.

    2. (12)(3456)

      The order is 4.

    3. (1234)(56789)

      The order is 20.

  4. What is the order of αβ, if α and β are disjoint cycles of 4 and 6, respectively? (Explain why. Use the fact that disjoint cycles commute.)

    First, since α and β are disjoint, αmβn = e only when αm = e and βn = e.

    Next, since α and β are disjoint cycles, they commute, hence (αβ)p = αpβp.

    Since the order of α is 4, α4 = α8 = α12 = e (by Exercise 1). Since the order of β is 6, β6 = β12 = e (by Exercise 1).

    (αβ)12 = α12β12 = e∘e = e, so the order of αβ is 12.

  5. What is the order of αβ if α and β are disjoint cycles of lengths r and s, respectively?

    The order of αβ must be a multiple of r in order for the product of the αs to be the identity. Similarly, it must be a multiple of s in order for the product of the βs to be the identity. Therefore, the order of αβ must be a multiple of both r and s.

    Since the order of a permutation, π, is defined as the least positive integer n such that πn = e, the order of αβ is therefore lcm(r,s).
