Problem A — Examples of Abelian Groups

For each of the following operations (on R), determine whether or not it is an Abelian group.

  1. x*y = x+y+k, on the set R

    The operation is commutative, by the commutativity of addition.

    (x*y)*z = (x+y+k)*z = x+y+z+2k
    x*(y*z) = x*(y+z+k) = x+y+z+2k
    The operation is associative.

    Solving e*y = y gives: e+y+k = y; e = -k
    Solving x*e = x gives: x+e+k = x; e = -k
    Therefore, -k is the identity element.

    Solving y'*y = -k gives: y'+y+k = -k; y' = -(y+2k)
    Solving x*x' = -k gives: x+x'+k = -k; x' = -(x+2k)
    Therefore, each element x has an inverse -(x+2k)

    This set and operation form an Abelian group.

  2. x*y = xy/2, on the set R-{0}

    The operation is commutative, by the commutativity of multiplication.

    (x*y)*z = (xy/2)*z = xyz/4
    x*(y*z) = x*(yz/2) = xyz/4
    The operation is associative.

    Solving e*y = y gives: ye/2 = y; e = 2
    Solving x*e = x gives: xe/2 = x; e = 2
    Therefore, 2 is the identity element.

    Solving y'*y = 2 gives: y'y/2 = 2; y' = 4/y
    Solving x*x' = 2 gives: xx'/2 = 2; x' = 4/x
    Therefore, each element x has an inverse 4/x.

    This set and operation form an Abelian group.

  3. x*y = x+y+xy, on the set R-{-1}

    The operation is commutative, by the commutativity of addition and multiplication.

    (x*y)*z = (x+y+xy)*z = x+y+z+xy+xz+yz+xyz
    x*(y*z) = x*(y+z+yz) = x+y+z+yz+xy+xz+xyz
    The operation is associative.

    Solving e*y = y gives: e+y+ey = y; e+ey = 0; e(1+y)=0; e = 0
    Solving x*e = x gives: x+e+xe = x; e+xe = 0; e(1+x) = 0; e = 0
    Therefore, 0 is the identity element.

    Solving y'*y = 0 gives: y'+y+y'y = 0; y'(1+y) = -y; y' = -y/(1+y)
    Solving x*x' = 0 gives: x+x'+xx' = 0; x'(1+x) = -x; x' = -x/(1+x)
    Therefore, each element x has an inverse -(x+2k).

    This set and operation form an Abelian group.

  4. x*y = (x+y)/(xy+1) on the open interval (-1,1)

    The operation is commutative, by the commutativity of addition and multiplication.

    (x*y)*z = ((x+y)/(xy+1))*z = ((x+y)+z)/((x+y)z/(xy+1)+1) = (x+y+xyz+z)/(xz+yz+xy+1)
    x*(y*z) = x*((y+z)/(yz+1)) = (x+(y+z)/(yz+1))/(x(y+z)/(yz+1)+1) = (xyz+x+y+z)/(xy+xz+yz+1)
    The operation is associative.

    Solving e*y = y gives: (e+y)/(ey+1) = y; e+y = ey2+y; e(1-y2) = 0; e = 0
    Solving x*e = x gives: (x+e)/(xe+1) = x; x+e = x2e+x; e(1-x2) = 0; e = 0
    Therefore, 0 is the identity element.

    Solving y'*y = 0 gives: (y'+y)/(y'y+1) = 0; y'+y=0; y'=-y
    Solving x*x' = 0 gives: (x+x')/(xx'+1) = 0; x+x'=0; x'=-y
    Therefore, each element x has an inverse -x.

    This set and operation form an Abelian group.
