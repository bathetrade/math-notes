Problem F — Homomorphism and the Order of Elements

If f:G→H is a homomorphism, prove each of the following.

  1. For each element a ∈ G, the order of f(a) is a divisor of the order of a.

    Let |a| = n. Then, an = e. Since homomorphisms preserve products, it is also true that f(an) = e = (f(a))n. Since f(a)n = e, |f(a)| | n = |a|. ∎

  2. The order of any element b≠e in the range of f is a common divisor of |G| and of |H|.

    By Lagrange's Theorem, |b| ∣ |G|. By Exercise 1, |f(b)| ∣ |b|, so |f(b)| ∣ |G|. Back to Lagrange, |f(b)| ∣ |H|. ∎

  3. If the range of f has n elements, xn ∈ ker(f) ∀ x ∈ G.

    Since the range of a homomorphism is a group, the order of each element in the range of f divides n. Therefore, ∀ f(x), (f(x))n = e. Since homomorphisms preserve products, f(xn) = e. Therefore, xn ∈ ker(f). ∎

  4. Let m ∈ ℤ such that gcd(m, |H|) = 1. ∀ x ∈ G, if xm ∈ ker(f), x ∈ ker(f).

    If xm ∈ ker(f), then f(xm) = e. Since homomorphisms preserve products, (f(x))m = e. Therefore, |f(x)| ∣ m. Since gcd(m, |H|) = 1, gcd(|f(x)|, |H|) = 1. Because the order of an element divides the order of the group, this means that |f(x)| = 1, or f(x) = e. Therefore x ∈ ker(f). ∎

  5. Let the range of f have m elements. If a ∈ G has order n, where gcd(m,n) = 1, then a ∈ ker(f).

    |f(a)| ∣ |H| = m, by Exercise 2. |f(a)| ∣ |a| = n, by Exercise 1. Therefore, |f(a)| is a common divisor of m and n. Since gcd(m,n) = 1, |f(a)| = 1, which means that f(a) = e and a ∈ ker(f).

  6. Let p be prime. If ran(f) has an element of order p, then G has an element of order p.

    Let |f(a)| = p. Then, by Exercise 1, p ∣ |a|. Let |a| = kp. Then, e = akp = (ak)p, and |ak| = p. (|ak| cannot be less than p, because it must divide p since (ak)p = e.) ∎
