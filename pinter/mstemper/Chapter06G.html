Problem G — Some General Properties of Functions

For Exercises 1-3, let f:A→B and g:B→C be functions.

  1. Prove that if g∘f is injective, f is injective.

    The proof is by contradiction. If f is not injective, then ∃ p,q∈A such that f(p)=f(q). In that case, g(f(p))=g(f(q)), which would mean that g∘f is not injective. Therefore, since g∘f is injective, f must be as well.

  2. Prove that if g∘f is surjective, g is surjective.

    The proof is by contradiction. If g is not surjective, then ∃r∈C such that for no s∈B is it true that g(s)=r. In that case, there cannot be any t∈A such that g(f(t))=r, which means that g∘f is not surjective. Therefore, since g∘f is surjective, g must be as well.

  3. If f is injective and g is surjective, does that imply that g∘f is bijective?

    No, it does not. Let f:N→N be the injection f(x)=2x. Let g:N→N be the surjection g(x)=x. [g∘f](x)=2x, which is not a surjection on N, and therefore not a bijection.

  4. Let f:A→B and g:B→A be functions, with y=f(x) iff x=g(y). Prove that f is bijective and that g=f-1.

    The proof that f is injective is by contradiction. Assume that f is not injective. Then, ∃a,b∈A such that f(a)=f(b)=q. By the problem statement, this would mean that both g(q)=a and g(q)=b, which cannot be true. Therefore, f must be an injection.

    The proof that f is surjective is direct. For any c∈B, g(c)=y∈A, and we must then have f(y)=c.

    Since f is both injective and surjective, it is bijective.

    Since g(f(x))=x, g=f-1.
