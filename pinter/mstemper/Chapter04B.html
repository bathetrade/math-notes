Problem B — Rules of Algebra in Groups

Prove or disprove each of the following. Counter-examples may be taken from the six-element group of matrices in Chapter 3.

  1. x2 = e → x = e

    In a two element group, the element that is not the identity element must be its own inverse. Calling that element "x" gives x2 = e.

  2. x2 = a2 → x = a.

    From the previous example, the square of the non-identity element in a two-element group is e, and e = e2.

  3. (ab)2 = a2b2

    Since (ab)2 = abab, and a2b2 = aabb, this can only be true if abab = aabb.

    abab aabb
    a-1ababb-1 a-1aabbb-1 pre-multiply by a-1 and post-multiply by b-1
    ebae eabe inverses
    ba ab identity

    Therefore abab = aabb requires that ba =ab (that is, a and b commute). In the six-element group from Chapter 3, AB = C while BA = K.

  4. x2 =x → x = e

    x2 x
    xx x definition of x2
    xxx-1 xx-1 pre-multiply by x-1
    xe e inverse
    x e identity

  5. x∈G → ∃ y∈G such that y2 = x

    In the six-element group from Chapter 3, this is not true for elements A, C, or K.

  6. ∀ x∈ G and y∈G, ∃ z∈G such that y = xz.

    Let z = x-1y. Then xz = xx-1y = ey =y.
