Problem I — Conjugate Elements

If a ∈ G, a conjugate of a is any element of the form xax-1, where x ∈ G. (Also see Chapter 8, Problem E.)

For any element a ∈ G, the centralizer of a, Ca, is the set of all elements of G which commute with a. That is, Ca = {x ∈ G : xa = ax} = {x ∈ G : xax-1 = a}.

Prove each of the following:

  1. The relation "a is a conjugate of b" (a~b) is an equivalence relation on G.

    Reflexive Since e = e-1, a = eae = eae-1. Therefore, a~a and this relation is reflexive.

    Symmetric Assume that a~b. Then ∃ x ∈ G such that b = xax-1.

    b xax-1 a~b
    bx xax-1x post-multiply both sides by x
    bx xae inverses
    bx xa identity
    x-1bx x-1xa pre-multiply both sides by x-1
    x-1bx ea inverses
    x-1ax b identity
    x-1a(x-1)-1 b an element is the inverse of its inverse

    Therefore, a~b implies b~a and this relation is symmetric.

    Transitive Assume that a~b and b~c. Then ∃ x ∈ G such that a = xbx-1 and ∃ y ∈ G such that b = ycy-1. Replacing b in the first expression with ycy-1 gives a = xycy-1x-1, or a = (xy)c(xy)-1. Therefore, a~b and b~c implies a~c and this relation is transitive.

    Since this relation is reflexive, symmetric, and transitive, it is an equivalence relation.

  2. For any a ∈ G, Ca is a subgroup of G.

    If x ∈ Ca, then xax-1 = a.

    xax-1 a x ∈ Ca
    x-1xax-1 x-1a pre-multiply both sides by x-1
    eax-1 x-1a inverse
    ax-1 x-1a identity
    ax-1x x-1ax post-multiply both sides by x
    ae x-1ax inverse
    a x-1ax identity
    a x-1a(x-1)-1 an element is the inverse of its inverse

    Therefore, x ∈ Ca implies that x-1 ∈ Ca, and Ca is closed under inversion.

    If x,y ∈ Ca, then xa = ax and ya = ay.

    (xy)a x(ya) associativity
    x(ay) ya = ay
    (xa)y associativity
    (ax)y xa = ax
    a(xy) associativity

    Therefore, (xy) commutes with a, making (xy) ∈ Ca, and Ca is closed under multiplication.

    Since Ca is closed under inversion and multiplication, it is a subgroup.

  3. x-1ax = y-1ay iff xy-1 commutes with a (that is, xy-1 ∈ Ca).

    If If (xy-1) commutes with a, we have:

    x-1ax x-1axe identity
    x-1ax(y-1y) inverse
    x-1a(xy-1)y associativity
    x-1(xy-1)ay (xy-1) commutes with a
    (x-1x)y-1ay associativity
    ey-1ay inverse
    y-1ay identity

    Therefore, if (xy-1) commutes with a, x-1ax = y-1ay.

    Only If If x-1ax = y-1ay, we have:

    x-1ax y-1ay given
    xx-1ax xy-1ay pre-multiply both sides by x
    eax xy-1ay inverse
    ax xy-1ay identity
    axy-1 xy-1ayy-1 post-multiply both sides by y-1
    axy-1 xy-1ae inverse
    axy-1 xy-1a identity

    Therefore, if x-1ax = y-1ay, (xy-1) commutes with a.

  4. x-1ax = y-1ay iff Cax = Cay.

    In Exercise 3, we showed that x-1ax = y-1ay iff xy-1 ∈ Ca. In Problem E, Exercise 1, we showed that ab-1 ∈ H iff Ha = Hb.

    Combining these gives x-1ax = y-1ay iff Cax = Cay.

  5. There is a one-to-one correspondence between the conjugates of a and the cosets of Ca.

    By Exercise 4, two conjugates of a, such as x-1ax and y-1ay, are equal iff the cosets of Ca (Cax and Cay) are equal.

  6. The number of distinct conjugates of a is equal to (G : Ca).

    Since there is a one-to-one correspondence between the distinct cosets of Ca and the distinct conjugates of a, the number of distinct conjugates is the same as the number of cosets, which is the index, (G : Ca).
