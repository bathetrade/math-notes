Problem B — Examples of Homomorphisms of Infinite Groups

Prove that each of the following is a homomorphism, and describe its kernel.

  1. The function φ:F(R) → R given by φ(f) = f(0).

    The group operation on F(R) was not specified. I will assume that it is addition, not composition.

    To prove that φ is a homomorphism, let a,b ∈ F(R).

    φ(a) + φ(b) a(0) + b(0) definition of φ
    (a+b)(0) definition of addition of functions
    φ(a+b) definition of φ

    Since φ preserves "products" (which are sums in this case), it is a homomorphism.

    ker(φ) = {c ∈ F(R) : c(0)=0}.

  2. The function φ:D(R) → F(R) given by φ(f) = f'.

    To prove that φ is a homomorphism, let a,b ∈ D(R).

    φ(a) + φ(b) a' + b' definition of φ
    (a+b)' sum of derivatives is derivative of sum
    φ(a+b) definition of φ

    Since φ preserves "products", it is a homomorphism.

    ker(φ) = {c ∈ D(R) : c'=0}. In other words, all constant functions.

  3. The function f:R × R → R given by f(x, y) = x + y.

    Pinter did not specify the operation on R × R. I am assuming that it is (p,r) + (q,s) = (p+q, r+s).

    To prove that f is a homomorphism, let (p,r), (q,s) ∈ R × R.

    f(p,r) + f(q,s) (p+r) + (q+s) definition of f
    (p+q) + (r+s) associativity and commutativity of addition
    f(p+q, r+s) definition of f
    f((p,r) + (q,s)) definition of addition in domain of f

    Since f preserves "products", it is a homomorphism.

    ker(f) = {(x,-x) : x ∈ R}.

  4. The function f:R* → R+ given by f(x) = |x|.

    I am assuming that the group operations are multiplication in both the domain and the range of f.

    To prove that f is a homomorphism, let x,y ∈ R*.

    f(x) · f(y) |x| · |y| definition of f
    |x·y| property of absolute value
    f(x·y) definition of f

    Since f preserves products, it is a homomorphism.

    ker(f) = {-1, 1}.

  5. The function f:C* → R+ given by f(a+bi) = sqrt(a2 + b2).

    I am assuming that the group operations are multiplication in both the domain and the range of f.

    To prove that f is a homomorphism, let x=a1+b1i, y=a2+b2i.

    f(x) · f(y) sqrt(a12 + b12) · sqrt(a22 + b22) definition of f
    sqrt( (a12 + b12) · (a22 + b22) ) property of square root
    sqrt( a12a22 + a12b22 + b12a22 + b12b22 ) distributivity
    sqrt( (a12a22 - 2a1a2b1b2 + b12b22) + (b12a22 + 2a1a2b1b2 + a12b22) ) inverses
    sqrt( (a1a2 - b1b2)2 + (b1a2 + a1b2)2 ) factor
    f( (a1a2 - b1b2) + (b1a2 + a1b2)i ) definition of f
    f( (a1 + b1i) · (a2 + b2i) ) multiplication in C
    f(x · y) definitions of x,y

    Since f preserves products, it is a homomorphism.

    ker(f) = {(a+bi) : sqrt(a2 + b2) = 1}.

  6. Let G be the set of all 2x2 matrices

    A = 
    ab
    cd

    satisfying ad-bc ≠ 0. Let f:G → R* be given by f(A) = det(A) = ad-bc.

    I am assuming that the group operation in G is matrix multiplication and that the group operation in R* is multiplication. I am also assuming that G is the set of all 2x2 real matrices.

    To prove that f is a homomorphism, let

    X = 
    x1,1x1,2
    x2,1x2,2

    and

    Y = 
    y1,1y1,2
    y2,1y2,2

    det(X) = x1,1x2,2 - x1,2x2,1. det(Y) = y1,1y2,2 - y1,2y2,1.

    det(X)det(Y) = (x1,1x2,2y1,1y2,2 + x1,2x2,1y1,2y2,1) - (x1,1x2,2y1,2y2,1 + x1,2x2,1y1,1y2,2).

    Multiplying X and Y gives:

    XY = 
    x1,1y1,1 + x1,2y2,1 x1,1y1,2 + x1,2y2,2
    x2,1y1,1 + x2,2y2,1 x2,1y1,2 + x2,2y2,2
    det(XY) (x1,1y1,1 + x1,2y2,1)(x2,1y1,2 + x2,2y2,2) - (x1,1y1,2 + x1,2y2,2)(x2,1y1,1 + x2,2y2,1)
    (x1,1y1,1x2,1y1,2 + x1,1y1,1x2,2y2,2 + x1,2y2,1x2,1y1,2 + x1,2y2,1x2,2y2,2) - (x1,1y1,2 + x1,2y2,2)(x2,1y1,1 + x2,2y2,1)
    (x1,1y1,1x2,1y1,2 + x1,1y1,1x2,2y2,2 + x1,2y2,1x2,1y1,2 + x1,2y2,1x2,2y2,2) -
    (x1,1y1,2x2,1y1,1 + x1,1y1,2x2,2y2,1 + x1,2y2,2x2,1y1,1 + x1,2y2,2x2,2y2,1)
    (x1,1y1,1x2,2y2,2 + x1,2y2,1x2,1y1,2) - (x1,1y1,2x2,2y2,1 + x1,2y2,2x2,1y1,1)

    det(X)det(Y) = det(XY). Since f preserves products, it is a homomorphism.

    ker(f) = {A ∈ G : det(A) = 1}.
