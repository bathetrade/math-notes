Problem A — Isomorphism Is an Equivalence Relation Among Groups

  1. Let G be any group. If ε:G → G is the identity function, ε(x)=x, show that ε is an isomorphism.

    If ε(x1) = ε(x2), then x1 = x2, so ε is injective. If q ∈ G, then ε(q) = q, so ε is surjective. Since ε is injective and surjective, it is a bijection.

    Let a,b ∈ G. Since ε(a) = a, ε(b) = b, and ε(ab) = ab, ε(a)ε(b) = ε(ab).

    Since ε is a bijection and ε(a)ε(b) = ε(ab), ε is an isomorphism.

  2. Let G1 and G2 be groups, and f:G1 → G2 an isomorphism. Show that f-1:G2 → G1 is an isomorphsim.

    Since f is an isomorphism, it is a bijection. As shown at the end of Chapter 6, the inverse of a bijection is also a bijection. This means that f-1 is a bijection.

    Let a,b ∈ G1. Let f(a) = x and let f(b) = y. Then, x,y ∈ G2. Since f is an isomorphism, f(ab) = f(a)f(b) = xy.

    Since f-1 is a bijection and f-1(x)f-1(y) = f-1(xy), f-1 is an isomorphism.

  3. Let G1, G2, and G3 be groups, and let f:G1 → G2 and g:G2 → G3 be isomorphisms. Prove that g∘f:G1 → G3 is an isomorphism.

    Since f and g are isomorphisms, they are bijective functions. Since the composition of two bijective functions is bijective, g∘f is bijective.

    Let a,b ∈ G1. Let f(a) = x and let f(b) = y. Then, x,y ∈ G2. Since g is an isomorphism, g(x)g(y) = g(xy). Since f is also an isomorphism, f(ab) = f(a)f(b) = xy. This gives g(f(a))g(f(b)) = g(f(ab)), or g∘f(a)g∘f(b) = g∘f(ab).

    Since g∘f is bijective and g∘f(a)g∘f(b) = g∘f(ab), g∘f is an isomorphism.

Since the "is isomorphic to" relation is reflexive (Exercise 1), symmetric (Exercise 2), and transitive (Exercise 3), it is an equivalence relation and the title of this problem is justified.
